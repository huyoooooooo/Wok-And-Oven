## 食材选品
| 食材 | 门店选用 | 家常替代方案 |
| --- | --- | --- |
| 香干 | 衡山香干，烟熏味重、外韧里嫩 | "衡山香干"或"攸县香干"，选带烟熏味的 |
| 芹菜 | 竖批分条 → 横切小段 | 按你的观察执行，不用切斜段或滚刀块 |
| 辣椒 | 视频未说明品种，可能是二荆条/线椒 | 二荆条或线椒，香味足但辣度适中 |

关于衡山香干：它的特色是柴火和稻谷壳双次烘烤，切开后纹理清晰，既能吸附汤汁入味，又能保持嚼劲。

## 分步流程 + 时间/火候补全

1. 预处理（按你的观察）
    + 香干：切片（厚薄适中，太薄易烂，太厚不入味），备用。
    + 芹菜：竖批分条，再横切小段
    + 青辣椒：横刀分开，竖刀切小段

2. 煨制香干
    + 热锅下油，放入香干，为保证嫩度。用中小火煎煨。
    + 加入生抽和盐调味，煨入味。

    锅底"酱油有点多"的合理怀疑：厨房做菜，像生抽、老抽这类调料，直接下锅很容易在高温下焦化变苦或颜色过深。大概率是加了少量水，调成酱油水来煨，这样既入味，又能防止酱料在锅底烧干发苦。

    + 时间：中小火煨制约2-3分钟，让香干吸味即可，不需久煮。

3. 炒蔬菜（分锅操作）
    + 重新起锅，放油。下芹菜小段和青椒粒，用中大火翻炒至断生、香味溢出。
    + 加盐调味（先给蔬菜调味，后续合炒时整体咸淡更均匀）。

4. 合炒融合
    + 倒入煨好的香干，连同煨制时的汤底一起倒进去（你观察到"出锅时能看到汤底剩余"，说明汤底是特意保留的）。
    + 合炒时间：大火快速翻炒约30秒-1分钟，让香干和蔬菜的味道融合。
    + 出锅：汤汁不用收干，保留少量酱油色汤底，亮油出锅。

## 家庭制版

| 对比项 | 焯水版（你查到的） | 门店煨制版（你观察到的） |
| --- | --- | --- |
| 香干预处理 | 开水+酱油焯水3分钟 | 中小火+酱油水煨制2-3分钟 |
| 核心逻辑 | 焯水保嫩度+预入味 | 煨制保嫩度+让香干吸味 |
| 操作差异 | 先焯水，再起锅炒 | 先煨好，再分锅炒蔬菜，最后合炒 |
| 门店吻合度 | 大众做法 | 符合你观察到的门店操作 |

通常，家庭做法也建议焯水，焯水的目的是去除豆干自身的豆腥味
