# 蓝莓麦芬

## ✅ 材料清单（6个纸杯）

| 材料 | 用量 | 备注 |
| --- | --- | --- |
| 低筋面粉 | 120g | 过筛 |
| 无铝泡打粉 | 4g | 必须放，不能省 |
| 细砂糖 | 30g | 微甜，可按口味±5g |
| 盐 | 1g | — |
| 鸡蛋 | 1个（约50g） | 室温（提前从冰箱拿出） |
| 牛奶 | 80g | 室温（全脂/低脂均可） |
| 玉米油 | 30g | 无味植物油 |
| 新鲜蓝莓 | 80-100g | 洗净沥干，不用裹粉 |

## ⏰ 烤箱设置

提前 180℃ 进行预热

| 项目 | 参数 |
| --- | --- |
| 模式 | 上下火 |
| 温度 | 180℃ |
| 层位 | 中层 |
| 时间 | 20-25分钟（用牙签检测） |

## 👩‍🍳 制作流程

| 顺序 | 步骤 | 关键动作 |
| --- | --- | --- |
| 1 | 预热烤箱 | 180℃，提前开 |
| 2 | 混合干粉 | 低粉+泡打粉+糖+盐，手动搅匀（泡打粉一定要拌匀） |
| 3 | 混合湿料 | 鸡蛋+牛奶+玉米油，搅匀（鸡蛋牛奶必须室温，否则油会结块） |
| 4 | 干湿混合 | 湿料倒入干料，刮刀翻拌至无干粉立刻停（面糊粗糙有小疙瘩是正常的！） |
| 5 | 加入蓝莓 | 取一半蓝莓（约40-50g）倒入面糊，轻翻2-3下（破了没关系） |
| 6 | 装杯 | 面糊用勺子舀入纸杯，7分满 |
| 7 | 表面蓝莓 | 剩余蓝莓放在表面，轻轻按进一半（防止烤后掉出） |
| 8 | 烘烤 | 放入预热好的烤箱中层，180℃烤20-25分钟 |
| 9 | 检查熟度 | 牙签插入麦芬中心最厚处，拔出来没有湿面糊就是熟了 |
| 10 | 出炉 | 烤盘取出，放5分钟后移到烤架，彻底放凉再密封 |

## ⚠️ 关键注意点

| 注意点 | 原因 |
| --- | --- |
| 不要搅拌过度 | 面粉起筋→麦芬变硬、不松软。面糊粗糙有小疙瘩是正常的 |
| 牛奶鸡蛋必须室温 | 冷藏的会让玉米油结块，面糊搅不匀 |
| 泡打粉不能省 | 麦芬不靠打发，全靠泡打粉膨胀 |
| 装杯7分满 | 太满会溢，太少没有蘑菇头 |
| 表面蓝莓要按进一半 | 不按的话，烤完蓝莓会掉出来 |
| 出炉别急着吃 | 太烫蓝莓会爆浆烫嘴，放凉10-15分钟再吃 |

## 📦 保存

| 方式 | 时间 | 方法 |
| --- | --- | --- |
| 室温密封 | 2-3天 | 彻底放凉后装密封盒 |
| 冷冻 | 1个月 | 密封袋，吃之前烤箱150℃复热5分钟 |
| 冷藏 | ❌ 不推荐 | 会变硬 |

## 疑问

1. 无铝泡打粉

  麦芬面糊不经过发酵，也没有打发蛋白来撑起结构，膨胀全靠泡打粉受热后释放二氧化碳气体。没有它，烤出来就是一块实心的面疙瘩。

  烘焙店和超市都能买到

2. 细砂糖能不能换成白砂糖或糖霜？

  通常无特殊说明的细砂糖都可以替换糖霜，家用白砂糖注意颗粒度，如果太粗，记得研磨。而如果标注糖霜则一般不能反向替代，糖霜中包含3%的玉米淀粉，在烘焙中会有其他作用。家用中也可自行调配。
