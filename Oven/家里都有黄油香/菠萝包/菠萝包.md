# 菠萝包

## ✅ 配方（6个菠萝包）

+ **【面团部分】**

    | 材料 | 克重 | 备注 |
    | --- | --- | --- |
    | 高筋面粉 | 250g | 蛋白质≥12% |
    | 细砂糖 | 30g | — |
    | 盐 | 3g | — |
    | 耐高糖干酵母 | 3g | 金色包装 |
    | 全蛋液 | 50g（约1个） | 冷藏 |
    | 牛奶 | 125g | 冷藏，预留10g调整 |
    | 无盐黄油 | 25g | 室温软化，后加 |

+ **菠萝皮（新增）**

    | 材料 | 克重 | 备注 |
    | --- | --- | --- |
    | 无盐黄油 | 40g | 室温充分软化 |
    | 糖粉 | 35g | 不要用细砂糖 |
    | 全蛋液 | 15g（约1/3个） | 室温 |
    | 低筋面粉 | 70g | 过筛 |
    | 奶粉 | 10g | 可选，增加奶香 |

+ **表面装饰（可选）**

    | 材料 | 克重 |
    | --- | --- |
    | 全蛋液 | 适量（刷菠萝皮用） |

## 👨‍🍳 制作流程（分阶段）

1. **阶段一：面团制作（和你做蒜香面包一样）**

    + 水合：面粉+糖+牛奶+蛋液混合成团，冷藏1-2小时（成团即可，无需光滑）
    + 揉面：取出水合面团，加盐揉匀 → 加酵母揉匀 → 加黄油揉至厚膜（拉出厚膜，破洞边缘呈锯齿状）
    + 一发：28-30℃，发酵至**2倍大**（手指戳洞，不回缩不塌陷）

2. **阶段二：菠萝皮制作（一发期间做）**

    + 黄油软化至膏状（手指轻按能出坑）
    + 加入糖粉，用刮刀碾压搅拌均匀（不需要打发，拌匀即可）
    + 分次加入蛋液，每次搅匀再加下一次
    + 筛入低筋面粉和奶粉，用刮刀翻拌成团（不要过度揉，成团即停）
    + 包上保鲜膜，放冰箱冷藏20-30分钟（让菠萝皮变硬，方便后续整形）

3. **阶段三：整形**

    + 一发结束后，轻拍排气，分割成6份，滚圆，盖保鲜膜松弛15分钟
    + 取出冷藏的菠萝皮，分割成6份（每份约20-22g），搓圆
    + 关键操作：取一份菠萝皮，放在两张保鲜膜或油纸之间，压扁/擀成圆形薄片（比面团略大一圈，约10-12cm直径）
    + 撕开上层保鲜膜，把菠萝皮盖在面团上，用手轻轻包住面团，让菠萝皮覆盖面团表面约80%（底部留一圈不包，露出面包体）
    + 用刮板或刀背在菠萝皮上划出格子纹路（横竖交叉，不要太深，划破表皮即可）

4. **阶段四：二发与烘烤**

    + 整形好的菠萝包放入烤盘，二发（35℃，40-50分钟），发至1.8-2倍大
    + 二发完成后，菠萝皮表面刷一层蛋液（刷在纹路上，颜色会更漂亮）
    + 烤箱提前预热 170℃（比蒜香面包低10℃，因为菠萝皮含糖量高，温度太高容易糊）
    + 放入烤箱中下层，170℃烤18-20分钟
    + 最后5分钟盯紧：如果上色太快，盖锡纸

## ⚠️ 关键注意点

+ 菠萝皮	
    - 黄油软化到位（膏状），但不要融化；
    - 成团后必须冷藏，否则太软无法擀开/包覆
+ 包皮	
    - 菠萝皮不要包满整个面团，底部留一圈（约1cm）不包，露出面包体，否则发酵时底部会被撑开
+ 划纹路	
    - 不要划太深，划破菠萝皮表面即可，划太深烤的时候会裂开变形
+ 烘烤
    - 170℃ 低温慢烤，因为菠萝皮含糖，高温会焦；
    - 如果表面已经上色但时间没到，盖锡纸
