# 抹茶奶酥贝果（液种版）

贝果面团含水低、手感偏硬，是正常的，不要额外加水。

## ✅ 配方（6个贝果）

+ **【液种】**（提前一晚准备）

    | 材料 | 克重 | 备注 |
    | --- | --- | --- |
    | 高筋面粉 | 75g | 蛋白质≥12% |
    | 水 | 75g | — |
    | 干酵母 | 1g | 即发干酵母 |

+ **【主面团】**

    | 材料 | 克重 | 备注 |
    | --- | --- | --- |
    | 高筋面粉 | 175g | 蛋白质≥12% |
    | 抹茶粉 | 8-10g | 过筛；颜色/味道想浓一点取上限 |
    | 细砂糖 | 15g | — |
    | 盐 | 4g | 与酵母分开放 |
    | 干酵母 | 1.5g | 即发干酵母 |
    | 水 | 80-85g | 先放 80g，视吸水再补 |

+ **【抹茶奶酥馅】**（6个量，每个约 17g）

    | 材料 | 克重 | 备注 |
    | --- | --- | --- |
    | 无盐黄油 | 30g | 室温充分软化（手指按出坑） |
    | 糖粉 | 15g | 不要用细砂糖，会颗粒感 |
    | 全蛋液 | 11g | 室温 |
    | 奶粉 | 46g | 全脂奶粉，不要用牛奶替代 |
    | 抹茶粉 | 5g | 过筛 |

+ **【煮贝果糖水】**

    | 材料 | 克重 | 备注 |
    | --- | --- | --- |
    | 水 | 500g | — |
    | 细砂糖 | 25g | 不想上色太深可减到 15g 或不用 |

## 👨‍🍳 制作流程

1. **第一步：制作液种**

    + 液种材料混合均匀，盖保鲜膜
    + 室温发酵至表面布满气泡（约 2-4 小时），或冰箱冷藏 12-18 小时

2. **第二步：揉面**

    + 液种 + 主面团所有材料（盐和酵母分开放）混合
    + 揉至表面光滑、能拉出厚膜（破洞边缘呈锯齿状即可）
    + 面团含水量较低，稍硬是正常的，不要额外加水

3. **第三步：分割、松弛**

    + 面团均分成 6 份（每份约 70-75g），滚圆
    + 盖保鲜膜，室温松弛 20-30 分钟

4. **第四步：制作抹茶奶酥馅（松弛期间做）**

    + 软化黄油 + 糖粉，搅打至顺滑发白
    + 分次加入蛋液，每次搅匀再加下一次
    + 筛入奶粉和抹茶粉，翻拌成团备用
    + 太软就冷藏 10 分钟再操作

5. **第五步：整形与包馅**

    + 取一个松弛好的面团，擀成长方形（约 15cm 长）
    + 奶酥馅约 17g/个，铺在面片中间（也可用保鲜膜夹住擀开再铺）
    + 从上往下卷紧，收口捏紧
    + 长条轻轻搓长至约 20-22cm
    + 一头压扁擀成小扇形，包住另一端成圆圈，**接口一定要捏紧**

6. **第六步：最终发酵（二发）**

    + 整好形的贝果放油纸上，约 30℃ 发酵 30 分钟左右
    + 发至约 1.5 倍大；按压表面能缓慢回弹即可

7. **第七步：煮贝果（关键）+ 烘烤**

    + 烤箱提前预热至 200℃
    + 糖水煮到冒小泡（约 90℃），不要沸腾
    + 贝果每面各煮 15-20 秒，捞出沥干，放烤盘
    + 200℃ 烤 16-18 分钟；最后 5 分钟盯色，满意后可盖锡纸（抹茶面包容易上色）

## 注意事项

+ 松弛要到位：
    - 擀卷时回缩厉害 = 松弛不够，盖保鲜膜再等 5-10 分钟，别硬擀
+ 收口捏紧：
    - 卷紧，收口和接口捏死，否则煮的时候易散、馅易漏
+ 煮的时间：
    - 每面 15-20 秒即可；煮太久外皮过韧，内部膨胀也会受影响
+ 奶酥馅：
    - 黄油必须软化到位，否则颗粒感；不要过度搅拌，以免馅变稀
+ 烘烤：
    - 奶酥含糖 + 抹茶易烤黄，最后几分钟守着，必要时盖锡纸
