# Garlic Bread Roll或蒜香橄榄包。

操作手法：

面团整成橄榄形→发酵→在中间划一刀深口→挤上蒜蓉黄油酱→烘烤。酱汁会在烘烤时渗进切口里，表面金黄酥脆，内部柔软。

## 配方（6个橄榄包，适合28×28烤盘一炉）

+ **【面团部分】**

    | 材料 | 克数 | 备注 |
    | --- | --- | --- |
    | 高筋面粉 | 250g | 蛋白质含量12%以上 |
    | 细砂糖 | 30g | 不要减太多，影响上色和发酵 |
    | 盐 | 3g | 与酵母分开放 |
    | 耐高糖干酵母 | 3g | 即发干酵母，不是鲜酵母 |
    | 全蛋液 | 50g（约1个中等鸡蛋） | 冷藏蛋需回温 |
    | 牛奶 | 125g | 预留10g视吸水性调整 |
    | 无盐黄油 | 25g | 室温软化，后油法加入 |

+ **【蒜蓉黄油酱】**

    | 材料 | 克数 | 备注 |
    | --- | --- | --- |
    | 无盐黄油 | 30g | 室温充分软化（手指能轻松按出坑） |
    | 蒜头 | 15g（约4-5瓣） | 必须挤成泥，不能切碎 |
    | 盐 | 1g | 酱里调味用 |
    | 干欧芹碎/香葱碎 | 适量 | 可选，增色提香 |

+ **【表面装饰（可选）】**

    - 全蛋液少许（刷表面用，让面包烤出来亮亮的）
    - 干欧芹碎少许（出炉后撒）

## 👨‍🍳 制作过程（按时间线拆解）

1. **揉面（二选一）**

    **直接揉面（后油法）：**

    + 准备酵母液：取配方中一半的牛奶（**约60-65g，温度30℃左右**，夏季推荐用常温或微凉的牛奶化酵母），加入全部酵母（3g），搅拌一下，**静置5-10分钟**，表面出现泡沫即可
    + 混合干性材料：揉面盆中放入高筋面粉、细砂糖、盐，用筷子或手抽搅拌均匀，让盐和糖在面粉中分散开
    + 混合成团：将激活好的酵母液、剩余的牛奶（约60-65g）、全蛋液，全部倒入干性材料中。用筷子搅成絮状，然后上手揉成团
    + 手揉或厨师机揉到面团**光滑、不粘手、能拉出厚膜**（破洞边缘呈锯齿状，不需要透指纹的手套膜）
    + 加入软化的25g黄油，继续揉至黄油完全吸收、直至再次光滑不粘手，能拉出均匀的厚膜即可

    关键判断：面团温度控制在26-28℃最佳（夏季用冰牛奶或冰鸡蛋，避免面温过高）。

    直接法：先出膜 → 加黄油 → 再出膜

    **水合法：**

    + 面粉 + 糖 + 牛奶 + 蛋液，混合成团，盖保鲜膜放冷藏（30分钟~2小时）
    + 取出面团，撒入盐，揉匀；加入酵母，揉匀（注意先后顺序）
    + 加入软化的25g黄油，继续揉至黄油完全吸收、面团表面光滑不粘手，能拉出均匀的厚膜即可

    水合法：加盐、酵母 → 加黄油 → 直接揉到出膜

2. **第一次发酵（一发）**

    + 面团收圆，放入抹了薄油的盆中，盖保鲜膜
    + 温度28℃，湿度75%左右，发酵至2倍大
    + 判断标准：手指沾干粉戳洞，洞口不回缩、不塌陷。回缩=没发够，塌陷=发过了

3. **排气、分割、滚圆、松弛**

    + 一发完成的面团取出，轻拍排气（用手掌从中间向四周轻压，不要用力揉）
    + 分割成6等份（每份约75-80g）
    + 分别滚圆，盖保鲜膜松弛15-20分钟（这一步不能省，否则整形时面团回缩）

    **一发戳洞看定形**

4. **整形（橄榄形）**

    + 取一个松弛好的面团，光面朝下，擀成牛舌形（约15×8cm）
    + 从上往下卷起（像卷铺盖一样），卷紧，收口处一定要捏紧
    + 两端稍微搓尖，整成橄榄形（中间鼓、两头尖）
    + 收口朝下，放入铺了油纸的烤盘，每个之间留足间距（发酵后会膨胀）

5. **第二次发酵（二发）**

    + 温度35℃，湿度75-80%，发酵至1.8-2倍大
    + 判断标准：轻轻按压面包侧面，缓慢回弹。按压不回弹=发过了，快速弹回=还没发够

    **二发按压看回弹**

6. **划刀口（关键操作）**

    + 用锋利的刀片（刮眉刀片最佳），在面包正中间竖着划一刀
    + 深度约1-1.5cm（要划到能看见里面面团的切口，但不要划穿底部）
    + 动作要快、准、利落，不要来回拉锯（会破坏面团）

7. **挤蒜蓉黄油酱 + 烘烤**

    + 烤箱提前预热：180℃ 上下火（实际温度，建议用烤箱温度计校准）
    + 在划好的切口里挤入/填入蒜蓉黄油酱，尽量填满切口，表面也可以薄薄刷一层
    + 烘烤时间：180℃ 烤 15-18分钟，至表面金黄、蒜香浓郁
    + 判断熟度：轻轻按压面包侧面，回弹好就是熟了。如果表面上色过快，可以中途盖锡纸

8. **出炉**

    + 出炉后连烤盘震一下（排出热气）
    + 移到晾网架上冷却。趁热吃外酥里软最好吃，凉了外皮变软但依然香软

## 常见问题（建议重点看）

+ 问题1：面团揉了很久还是粘手
    - 原因：面粉吸水性不同或面温太高
    - 解决方法： 预留10g牛奶调整；面温太高时把面团摊平冷藏10分钟再揉
+ 问题2：一发后表面有大气泡
    - 原因：发酵时表面干了
    - 解决方法：盖保鲜膜或湿布，不要太严实但不能干
+ 问题3：整形时面团回缩
    - 原因：松弛时间不够
    - 解决方法：滚圆后盖保鲜膜，至少松弛15分钟
+ 问题4：划刀口时面团塌了
    - 原因：二发过头或刀不够锋利
    - 解决方法：二发勤看状态；用锋利刀片沾水再划，动作快
+ 问题5：切口挤酱后酱流得到处都是
    - 原因：酱太稀或挤太多
    - 解决方法：蒜蓉黄油酱应是膏状不要融化；挤好后轻按让酱渗入切口
+ 问题6：烤完底部有油渗出
    - 原因：黄油酱从切口流到烤盘
    - 解决方法：可在烤盘底垫油纸，烤盘稍倾斜或减少酱量
+ 问题7：蒜味发苦
    - 原因：蒜烤糊了或蒜泥不够细
    - 解决方法：蒜必须挤成泥不要碎粒；如果怕糊，可以先烤面包8分钟再取出抹酱（两步烤法）
+ 问题8：面包不够软
    - 原因：揉面不到位或烤太久
    - 解决方法：确保揉出厚膜；最后3分钟盯着烤箱，不要超时
+ 问题9：出炉后塌陷
    - 原因：没烤熟或二发过头
    - 解决方法：用竹签插入中心不带出面糊就是熟了；二发不要超过2倍
