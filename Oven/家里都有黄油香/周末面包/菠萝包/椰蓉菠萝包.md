# 椰蓉菠萝包

在菠萝包基础上增加椰蓉馅，酥皮做法与菠萝包一致，多一步包馅即可。

## ✅ 配方（6个椰蓉菠萝包）

+ **【面团部分】**（250g 高筋粉，与花环面包基本一致，加了少量奶粉）

    | 材料 | 克重 | 备注 |
    | --- | --- | --- |
    | 高筋面粉 | 250g | 蛋白质≥12% |
    | 奶粉 | 12g | 增加奶香，没有也可以不加 |
    | 细砂糖 | 25g | 不用减，面团本身不会太甜 |
    | 盐 | 2g | — |
    | 耐高糖干酵母 | 3g | 金色包装 |
    | 全蛋液 | 约30g（大半个鸡蛋） | 剩下的留着刷表面或做馅料 |
    | 牛奶 | 130g | 先放120g，预留10g |
    | 无盐黄油 | 18g | 后油法，室温软化 |

+ **椰蓉馅**（6个量，每个约30g）

    | 材料 | 克重 | 备注 |
    | --- | --- | --- |
    | 无盐黄油 | 45g | 室温软化到膏状 |
    | 糖粉 | 20g | 喜欢甜可以加到25g |
    | 全蛋液 | 20g | 室温，分次加入 |
    | 椰蓉 | 50g | 原味无糖椰蓉 |
    | 奶粉 | 25g | 没有可以换成等量椰蓉 |

+ **酥皮**（6个量，每个约35g）

    | 材料 | 克重 | 备注 |
    | --- | --- | --- |
    | 无盐黄油 | 35g | 室温软化 |
    | 糖粉 | 25g | — |
    | 全蛋液 | 15g | — |
    | 奶粉 | 15g | — |
    | 低筋面粉 | 65g | 做酥皮的核心材料 |

## 🍳 制作流程

1. **第一步：揉面**

    + 水合：面粉+糖+牛奶+蛋液混合成团，冷藏1-2小时（成团即可，无需光滑）
    + 揉面：取出水合面团，视干湿情况补加预留牛奶 → 加盐揉匀 → 加酵母揉匀 → 加黄油揉至厚膜（破洞边缘呈锯齿状）
    + 一发：28-30℃，发至**2倍大**（手指戳洞，不回缩不塌陷）

2. **第二步：准备椰蓉馅（一发期间完成）**

    + 黄油软化至膏状，加入糖粉搅匀
    + 分次加入蛋液，每次都搅匀再加下一次
    + 加入椰蓉和奶粉，翻拌均匀成团
    + 分成6份，搓圆，盖保鲜膜备用

3. **第三步：准备酥皮（一发期间做，二发前完成）**

    + 黄油软化至膏状，加入糖粉搅匀
    + 分次加入蛋液，每次搅匀再加下一次
    + 筛入低筋面粉和奶粉，翻拌成团（不要过度揉）
    + 分成6份，搓圆，放冰箱冷藏20-30分钟定型

4. **第四步：包馅、盖酥皮**

    + 一发好的面团分割成6份，滚圆，盖保鲜膜松弛15分钟
    + 取一个面团，压扁，像包包子一样包入椰蓉馅，收口捏紧，朝下放置
    + 取一份冷藏好的酥皮，放在两层保鲜膜或油纸之间，压扁/擀成圆形薄片（比面团略大一圈）
    + 把酥皮盖在面团上，轻轻包住面团表面（底部约1cm不用包）
    + 用刮板或刀背在酥皮上划出格子纹路（不要太深，划破表皮即可）

5. **第五步：二发与烘烤**

    + 摆上烤盘，二发（35℃，40-50分钟），发至1.8-2倍大
    + 二发完成后，酥皮表面刷一层蛋液
    + 烤箱提前预热170℃，中下层，烤18-20分钟
    + 最后5分钟盯紧：如果上色太快，盖锡纸

## ⚠️ 关键注意点

+ 牛奶用量
    - 先放120g，预留10g备用，根据面团状态决定是否补加
+ 椰蓉馅状态
    - 黄油软化到膏状即可，不用融化；馅料能捏成团就行
    - 如果太软，冷藏一会儿再包
+ 酥皮冷藏
    - 做好后一定要冷藏15-20分钟，否则太软会擀不开、包不住
+ 包酥皮手法
    - 酥皮包住面团表面即可，**底部留一圈约1cm不包**，让面包体在二发时有膨胀空间
+ 二发时间
    - 酥皮会轻微限制面团膨胀，二发可能比普通面包稍慢一点，**看状态不看时间**
    - 发到1.8倍大、按压缓慢回弹即可
+ 烘烤
    - 酥皮含糖量高，最后几分钟多看一眼
    - 上色满意就可以出炉，不要烤过头；上色太快盖锡纸
