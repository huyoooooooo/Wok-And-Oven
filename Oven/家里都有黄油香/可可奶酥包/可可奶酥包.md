# 可可奶酥面包

面团部分和蒜香面包大体一致，唯一新增的是“奶酥馅”和“整形手法”。

## 配方（6个花环/麻花面包）

+ **【面团部分】**

    | 材料 | 克重 | 备注 |
    | --- | --- | --- |
    | 高筋面粉 | 250g | 蛋白质≥12% |
    | 细砂糖 | 30g | — |
    | 盐 | 3g | — |
    | 耐高糖干酵母 | 3g | 金色包装 |
    | 全蛋液 | 50g（约1个） | 冷藏 |
    | 牛奶 | 125g | 冷藏，预留10g调整 |
    | 无盐黄油 | 25g | 室温软化，后加 |

+ **可可奶酥馅（新增）**

    | 材料 | 克重 | 备注 |
    | --- | --- | --- |
    | 无盐黄油 | 40g | 室温充分软化 |
    | 糖粉 | 25g | 不要用细砂糖，会颗粒感 |
    | 全蛋液 | 15g（约1/3个） | 室温 |
    | 奶粉 | 40g | 全脂奶粉，不要用牛奶替代 |
    | 可可粉 | 8g | 过筛，无糖烘焙用 |

    如果换成抹茶粉，用量改为 6g（抹茶粉比可可粉颜色更深、味道更浓，稍减量）。

+ **表面装饰（可选）**

    | 材料 | 克重 |
    | --- | --- |
    | 全蛋液 | 适量（刷表面用） |
    | 杏仁片/杏仁碎 | 适量（撒表面） |

## 👨‍🍳 制作流程

1. **第一步：水合（和蒜香面包一样）**

    + 面粉 + 糖 + 牛奶 + 蛋液，混合成团，冷藏水合1-2小时。
    + 取出后，加盐揉匀 → 加酵母揉匀 → 加黄油揉至厚膜。

2. **第二步：一发**

    + 28-30℃，发酵至2倍大。手指戳洞不回缩不塌陷。

3. **第三步：制作可可奶酥馅（一发期间或松弛时做）**

    + 黄油软化至膏状，加入糖粉，搅打至顺滑发白。
    + 分次加入蛋液，每次搅匀再加下一次。
    + 筛入奶粉和可可粉，用刮刀翻拌均匀成膏状。如果觉得太软，放冰箱冷藏10分钟再操作。

4. **第四步：排气、分割、松弛**

    + 一发完成后，轻拍排气，分割成6份，滚圆，盖保鲜膜松弛15分钟。

5. **第五步：整形（花环/麻花，二选一）**

    **花环造型（更简单）：**

    + 取一个面团，擀成长方形（约 20×30cm）。
    + 抹上可可奶酥馅（四周留 1cm 不抹）。
    + 从上往下卷起，卷紧，收口捏紧。
    + 把卷好的长条搓长至约 35-40cm。
    + 盘成圆形，两端捏在一起，形成一个环。
    + 用剪刀在环上均匀剪几刀（剪断表面露出馅料即可），然后把每个剪开的面团段往外翻开，形成花瓣状。

    **麻花造型（稍复杂）：**

    + 前3步同上（擀开 → 抹馅 → 卷起）。
    + 把卷好的长条搓长至约 35-40cm。
    + 用刀从中间纵向切开（顶端留 2cm 不切断）。
    + 两条面像扭麻花一样交叉扭在一起。
    + 两端收拢捏紧。

6. **第六步：二发**

    + 35℃ 湿温环境，约 40-50 分钟，发至 1.8-2 倍大。

7. **第七步：烘烤**

    + 二发完成后，表面刷一层全蛋液，撒上杏仁片。
    + 烤箱提前预热 180℃，中下层，烤 16-18 分钟。
    + 最后 3-5 分钟守在烤箱前，上色太快要盖锡纸（奶酥馅含糖，容易焦）。

## 注意事项

+ 奶酥馅：  
    - 黄油必须软化到位（手指按出坑），否则馅料有颗粒；
    - 不要过度搅拌，以免馅料变稀
+ 卷紧：  
    - 卷的时候每一下都压紧，收口一定要捏死，否则二发后容易露馅
+ 搓长： 
    - 搓的时候如果回缩厉害，说明松弛不够，再松5-10分钟再搓
+ 烘烤：
    - 奶酥含糖高，上色比普通面包快，最后几分钟盯紧，必要时盖锡纸
+ 可可粉：  
    - 可可粉含天然油脂，比抹茶粉更容易拌匀，过筛后直接和奶粉混合就行
