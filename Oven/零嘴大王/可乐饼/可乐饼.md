# 可乐饼（烤箱版）

外酥内软、土豆泥和肉糜的经典组合——裹上面包糠过三关，200℃ 烤箱烤到两面金黄，能还原八成以上油炸效果，更省事也少油。

## 📊 油炸可乐饼 vs 烤箱可乐饼

| 对比项 | 油炸可乐饼 | 烤箱可乐饼（本食谱） |
| --- | --- | --- |
| 外皮 | 更脆，香气更足 | 接近油炸，表面油光略逊 |
| 用油量 | 多，要炸 | 少，表面刷薄油即可 |
| 操作 | 油温控制，溅油 | 预热+翻面，更省事 |
| 结论 | 追求极致酥脆时用 | 日常家常足够，80% 以上油炸效果 |

## 🥩 食材清单（6 个中等大小可乐饼）

**馅料**

| 食材 | 用量 | 说明 |
| --- | --- | --- |
| 土豆（带皮称） | 约 400g（2 个中等大小） | 选黄心/面土豆，不要用红皮或新土豆 |
| 洋葱 | 80g（约半个） | 切成细碎末 |
| 猪肉糜/牛肉糜 | 120g | 肥瘦 2:8 最佳，太瘦会干 |
| 黄油（炒馅用） | 10g | 或普通食用油 |
| 盐 | 3g（约半小勺） | — |
| 黑胡椒粉 | 适量 | 提味用 |
| 细砂糖 | 2g | 中和洋葱涩味，可不放 |
| 马苏里拉芝士 | 40g（可选） | 放中间做流心馅，不放也可 |

**裹粉三件套（过三关）**

| 材料 | 用量 | 说明 |
| --- | --- | --- |
| 低筋面粉 | 约 30g | 普通面粉也行 |
| 鸡蛋 | 1 个 | 打散成蛋液 |
| 面包糠 | 约 50g | 建议用日式粗面包糠，更脆 |

**表面/烤盘用**

| 材料 | 用量 | 说明 |
| --- | --- | --- |
| 植物油/融化的黄油 | 约 10g | 刷表面用，防止烤干 |
| 烘焙油纸 | 1 张 | 防粘 |

## ⏰ 烤箱设置

提前 200℃ 进行预热

| 项目 | 参数 |
| --- | --- |
| 模式 | 上下火 |
| 温度 | 200℃ |
| 层位 | 中层 |
| 时间 | 15 分钟 → 翻面 → 再烤 5-8 分钟 |

## 👨‍🍳 操作流程

1. **第一步：处理土豆**

    + 土豆带皮蒸熟或煮熟（约 20-25 分钟，筷子能轻松穿透）
    + 趁热去皮，压成泥，尽量压细，不要有硬块
    + 摊开放凉（一定要凉透，否则后面不好整形）

2. **第二步：炒馅**

    + 锅中放黄油，中小火炒洋葱碎，炒到透明、发软、有香味（约 3-4 分钟）
    + 加入肉糜，炒散、炒到变色、没有红肉
    + 加盐、黑胡椒、糖，翻炒均匀后关火放凉（同样要凉透）

3. **第三步：混合馅料**

    + 把完全放凉的土豆泥和完全放凉的肉馅混合在一起，用手或刮刀拌匀
    + 如果加芝士，先不拌进去，留到最后包

4. **第四步：整形（最关键）**

    + 把混合好的馅料分成 6 份（每份约 80-90g），搓成圆球
    + 如果加芝士：压扁 → 中间放一小块马苏里拉 → 包起来捏紧
    + 整成椭圆饼状（约 2cm 厚），不要太薄，否则烤完偏干
    + **馅料太软/粘手**：先放冰箱冷藏 20 分钟再整形

5. **第五步：过三关（裹粉顺序不能乱）**

    + **第一关**：均匀裹上一层薄薄的低筋面粉，拍掉多余的粉
    + **第二关**：完全浸入蛋液，让表面全部沾湿
    + **第三关**：滚上面包糠，用手轻轻压一下，让面包糠粘牢

6. **第六步：烘烤**

    + 烤盘铺油纸，放上可乐饼，表面刷一层薄油
    + 放入预热好的烤箱中层，200℃ 上下火
    + 烤 15 分钟 → 翻面 → 再烤 5-8 分钟，直到两面金黄酥脆

## ✅ 成品标准

+ 外皮：两面金黄酥脆，面包糠上色均匀
+ 内里：土豆泥绵软，肉馅湿润不散
+ 口感：外酥内软，咬开有土豆和肉的风味融合
+ 加芝士版：中间有流心拉丝效果

## ⚠️ 翻车预警

| 翻车现象 | 原因 | 改进 |
| --- | --- | --- |
| 馅料变稀、不成形、烤完塌 | 土豆泥没凉透就混合，会出水 | 土豆泥和炒馅都要凉透再混合 |
| 整形时粘手、包不住 | 馅料太软 | 放冰箱冷藏 20-30 分钟再整形 |
| 面包糠烤完发白、不金黄 | 油刷少了或烤的时间不够 | 表面一定要刷油，烤到上色再出炉 |
| 烤完偏干 | 馅料水分不足 | 馅料里可以加一点点牛奶或奶油（约 10g） |
| 一烤就散了 | 土豆泥压不够细或馅料没压实 | 整形时用力压紧，裹粉后也轻轻压实 |

## 📦 保存

| 方式 | 时间 | 方法 |
| --- | --- | --- |
| 室温 | 当天吃完 | 出锅放凉后密封，不建议久放 |
| 冷藏 | 1-2 天 | 密封盒存放，复热后外皮会变软 |
| 冷冻 | 约 1 个月 | 生坯或熟坯均可冷冻；熟坯 180℃ 复热 8-10 分钟 |
| 复热建议 | — | 烤箱或空气炸锅比微波效果更好，能保持部分酥脆 |

基础版做顺了，馅料可以换成虾仁、玉米或咖喱味来变花样。
