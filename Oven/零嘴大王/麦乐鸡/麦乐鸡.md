# 麦乐鸡（烤箱版）

鸡胸肉打成泥、加土豆泥和面包糠顺一个方向搅到起劲，200℃ 烤箱烤到表面金黄——不用油炸，也能做出 Q 弹不松散、接近市售麦乐鸡的口感。

## 📊 油炸麦乐鸡 vs 烤箱麦乐鸡

| 对比项 | 油炸麦乐鸡 | 烤箱麦乐鸡（本食谱） |
| --- | --- | --- |
| 外皮 | 更酥，香气更足 | 表面金黄，酥松感略逊 |
| 用油量 | 多，要炸 | 少，表面刷薄油即可 |
| 操作 | 油温控制，溅油 | 预热+翻面，更省事 |
| 结论 | 追求极致酥脆时用 | 日常家常足够，少油更友好 |

## 🥩 食材清单（8-10 块）

| 食材 | 用量 | 说明 |
| --- | --- | --- |
| 鸡胸肉 | 250g（约 1 块） | 去筋膜，切小块 |
| 土豆 | 100g（约半个） | 黄心土豆，蒸熟压泥 |
| 鸡蛋 | 1 个 | 全蛋液 |
| 洋葱 | 30g（约 1/4 个） | 切成极细的碎末 |
| 盐 | 3g | — |
| 黑胡椒粉 | 适量 | — |
| 大蒜粉 | 2g（可选） | 增加风味，没有可不放 |
| 面包糠 | 30g | 让口感更接近市售麦乐鸡，增加酥松感 |

**表面/烤盘用**

| 材料 | 用量 | 说明 |
| --- | --- | --- |
| 植物油/融化的黄油 | 约 10g | 刷表面用，防止烤干 |
| 烘焙油纸 | 1 张 | 防粘 |

## ⏰ 烤箱设置

提前 200℃ 进行预热

| 项目 | 参数 |
| --- | --- |
| 模式 | 上下火 |
| 温度 | 200℃ |
| 层位 | 中层 |
| 时间 | 12-15 分钟 → 翻面 → 再烤 5-8 分钟 |

## 👨‍🍳 操作流程

1. **第一步：处理土豆**

    + 土豆去皮切片，蒸熟（约 15 分钟），趁热压成泥
    + 彻底放凉备用（一定要凉透，否则后面混合会让肉质变硬）

2. **第二步：处理鸡肉**

    + 鸡胸肉切小块，用绞肉机打成肉泥（或用刀剁成细腻的肉糜）

3. **第三步：处理洋葱**

    + 洋葱切成极细的碎末（越细越好，颗粒太大影响口感）

4. **第四步：混合搅拌（关键）**

    + 在一个大碗里，放入鸡肉泥、土豆泥、洋葱碎、鸡蛋、面包糠、盐、黑胡椒、大蒜粉
    + 顺一个方向搅拌，约 2-3 分钟，直到肉泥变得粘稠、有弹性
    + **判断标准**：搅拌不够会松散，这是麦乐鸡「Q 弹」的关键

5. **第五步：整形**

    + 手上沾一点水（防粘），取约 40-45g 肉泥，捏成扁椭圆形的块状
    + 厚度约 1.5cm，宁可薄一点，不要厚

6. **第六步：烘烤**

    + 烤盘铺油纸，放上鸡块，表面刷一层薄油
    + 放入预热好的烤箱中层，200℃ 上下火
    + 烤 12-15 分钟 → 翻面 → 再烤 5-8 分钟，直到表面金黄上色即可

## ✅ 成品标准

+ 外观：椭圆块状，表面金黄上色
+ 口感：Q 弹不松散，外略酥内嫩
+ 味道：咸鲜适口，蘸番茄酱就很好
+ 没有大蒜粉：成品就是原味麦乐鸡，不影响主体风味

## ⚠️ 翻车预警

| 翻车现象 | 原因 | 改进 |
| --- | --- | --- |
| 成品不嫩、肉质偏硬 | 土豆泥没凉透就混合，肉泥温度升高 | 土豆泥一定要凉透再混合 |
| 烤完散开、没弹性 | 肉泥搅拌不够 | 顺一个方向搅拌 2-3 分钟，直到粘稠起劲 |
| 中间烤不透 | 鸡块太厚 | 厚度控制在 1.5cm 左右 |
| 烤完偏干 | 烤的时间过长 | 表面金黄就出炉，不用烤到焦黄 |
| 整形时粘手 | 肉泥水分大 | 手上沾点水再捏 |

## 💡 变化版

| 口味 | 做法 |
| --- | --- |
| 芝士麦乐鸡 | 中间包一小块马苏里拉芝士 |
| 玉米麦乐鸡 | 加入 30g 玉米粒（罐装沥干水分） |
| 香草麦乐鸡 | 加入 2g 干香草碎（迷迭香/百里香） |

## 📦 保存

| 方式 | 时间 | 方法 |
| --- | --- | --- |
| 室温 | 当天吃完 | 出锅放凉后密封，不建议久放 |
| 冷藏 | 1-2 天 | 密封盒存放，复热后口感略变韧 |
| 冷冻 | 约 1 个月 | 生坯或熟坯均可冷冻；熟坯 180℃ 复热 8-10 分钟 |
| 复热建议 | — | 烤箱或空气炸锅比微波效果更好 |
