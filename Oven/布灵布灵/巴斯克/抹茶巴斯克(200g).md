# 抹茶巴斯克（4寸 / 200g）

一整盒奶油奶酪刚好用完；茶香和芝士能平衡，不容易苦涩。

## ✅ 配方（4寸）

| 材料 | 克重 | 备注 |
| --- | --- | --- |
| 奶油奶酪 | 200g | 一整盒，提前 2 小时室温软化 |
| 细砂糖 | 45-50g | 抹茶微苦，这个量刚好平衡 |
| 抹茶粉 | 10-12g | 烘焙级（五十铃/若竹），颜色和风味更好 |
| 鸡蛋 | 2个（约 100-110g） | 常温（提前回温约 1 小时） |
| 淡奶油 | 130-140g | 约半盒（250ml 装），常温 |
| 玉米淀粉 | 5-7g | 过筛后加入 |
| 热水（70℃） | 20-25g | 调抹茶糊用，不能用开水 |

## ⏰ 烤箱设置

提前 220-230℃ 预热

| 项目 | 参数 |
| --- | --- |
| 模式 | 上下火 |
| 温度 | 220-230℃ |
| 模具 | 4 寸，铺好油纸 |
| 时间 | 20-24 分钟（建议从 18 分钟开始观察） |

## 👨‍🍳 制作流程

1. **第一步：准备**

    + 奶油奶酪提前 2 小时取出，软化到手指能轻松按出坑
    + 鸡蛋和淡奶油提前回温到常温
    + 烤箱提前预热 220-230℃
    + 4 寸模具铺好油纸（可裁剪，也可直接把烘焙纸捏皱铺进去）

2. **第二步：调抹茶糊（关键）**

    + 抹茶粉过筛，加入 70℃ 左右热水，搅成顺滑无颗粒的膏状
    + 分 2-3 次加入淡奶油（每次约 10g），搅匀成顺滑的抹茶奶油液
    + 水温不要超过 80℃，否则颜色发黄、容易发苦

3. **第三步：制作奶酪糊**

    + 软化好的奶油奶酪用刮刀压拌到顺滑无颗粒
    + 细砂糖和玉米淀粉混合均匀后加入，搅拌到顺滑
    + 鸡蛋打散，分 3-4 次加入，每次完全融合再加下一次
    + 加入剩下的淡奶油，拌匀

4. **第四步：混合与过筛**

    + 抹茶奶油液倒入奶酪糊，搅拌均匀
    + 面糊过筛 1-2 次（去掉抹茶颗粒和奶酪块，成品更细腻）
    + 倒入模具，轻震两下排出气泡

5. **第五步：烘烤**

    + 220-230℃ 烤 20-24 分钟；4 寸体积小，从 18 分钟开始观察
    + 判断标准：表面微微焦黄，中心轻微晃动（像果冻）即可出炉
    + 不要烤到完全凝固，余温会继续焖熟

6. **第六步：冷藏定型**

    + 出炉后彻底放凉（约 1 小时）
    + 连模具包好保鲜膜，冷藏至少 4 小时（最好过夜）

## 注意事项

+ 奶油奶酪软化：
    - 提前 2 小时取出，软化到手指能轻松按出坑
+ 鸡蛋 / 淡奶油回温：
    - 必须常温，否则容易油水分离
+ 抹茶粉结块：
    - 先用 70℃ 热水调成膏状；最后过筛可以挽救
+ 烤过头：
    - 4 寸从 18 分钟开始观察，中心晃动就出炉
+ 淡奶油剩余：
    - 密封冷藏 3 天内用完，或冷冻做其他用途
