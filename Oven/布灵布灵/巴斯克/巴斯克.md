# 巴斯克蛋糕

巴斯克按理来说全程应该是巴斯克芝士蛋糕，在甜品店常见的、外表“黑乎乎”的蛋糕。

1990年由西班牙圣塞瓦斯蒂安的La Viña餐厅首创的。

“烧焦”是特色：在220-240度的高温下快速烘烤，造就了标志性的焦黑外皮和湿润绵密的流心内馅，反差感极强。

原料极简：主要由奶油奶酪、鸡蛋、淡奶油和少量糖、面粉制成，芝士风味非常纯粹浓郁。制作也非常简单。

## 材料指南

1. 🧀 奶油奶酪（核心骨架）
    + 作用：蛋糕的绝对主体，负责提供浓郁的奶香、醇厚的风味，以及标志性、略带绵密扎实的口感。
    + 原理：奶油奶酪本身富含脂肪和蛋白质，在烘烤后凝固，构成了蛋糕的主体结构。巴斯克蛋糕的味道 70%以上取决于奶油奶酪的品质。

2. 🥚 鸡蛋（粘合剂与膨松剂）
    + 作用：鸡蛋是连接所有材料的桥梁，同时提供蓬松感。
    + 原理：
        - 蛋黄中的卵磷脂是天然的乳化剂，把油性的奶油奶酪和水性的淡奶油牢牢地“粘”在一起，让面糊顺滑不分离。
        - 蛋白在烘烤时会凝固并锁住空气，让蛋糕内部形成标志性的、软心甚至流心的质地。

3. 🥛 淡奶油（软化剂与湿润剂）
    + 作用：它的任务是稀释面糊，增加流动性，并注入额外的奶香。
    + 原理：若无淡奶油，面糊会像面团一样厚重。淡奶油中的水分在烘烤时变成水蒸气，使得蛋糕不干硬并呈现湿润、绵密、入口即化的效果。

4. 🌾 玉米淀粉（稳定剂与固化剂）
    + 作用：负责吸收多余水分，帮助蛋糕定型。
    + 原理：淀粉在高温下会糊化，吸收面糊里多余的水分，防止蛋糕在出炉后塌陷得太厉害。它让蛋糕在拥有流心质感的同时，依然能切出整齐的块状，而不是一盘散沙。

5. 🍬 细砂糖（不止是甜味）
    + 作用：它带来的远不止是甜。
    + 原理：
        - 保水：糖能吸附水分，让蛋糕内部保持湿润，延缓老化变干。
        - 上色：糖在高温下发生美拉德反应，标志性的焦黑色外皮，主要功劳就是糖。
        - 结构：糖能打断蛋白质的凝固过程，让蛋糕口感更柔软，不至于太硬。

>通俗的理解，做巴斯克蛋糕，就像盖房子。
>+ 奶油奶酪是决定房子风格的建筑师。
>+ 鸡蛋是把所有砖石粘在一起的水泥。
>+ 淡奶油是让房子不干燥的加湿器。
>+ 玉米淀粉是让房子不摇晃的地基钢筋。  
>+ 糖是给房子刷漆和装饰的油漆工，让它好看又好吃。
>+ 抹茶粉等是决定室内装修风格的软装设计师
