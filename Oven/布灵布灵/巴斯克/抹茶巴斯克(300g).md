# 抹茶巴斯克（6寸 / 300g）

综合几个配方的共同点，选了一个比较均衡的比例；茶香和芝士能平衡好，不容易苦涩。

## ✅ 配方（6寸）

| 材料 | 克重 | 备注 |
| --- | --- | --- |
| 奶油奶酪 | 300g | 提前 2 小时室温软化 |
| 细砂糖 | 60-70g | 抹茶微苦，糖太少压不住 |
| 抹茶粉 | 18-20g | 烘焙用抹茶粉（五十铃/若竹），颜色和风味更好 |
| 鸡蛋 | 3个（全蛋，约 150g） | 常温，提前从冰箱取出 |
| 淡奶油 | 200-220g | 常温，动物奶油（安佳/蓝风车/总统） |
| 玉米淀粉 | 8-10g | 过筛后加入 |
| 热水（70℃） | 40-50g | 调抹茶糊用，不能用开水（会发苦） |

## ⏰ 烤箱设置

提前 230℃ 预热

| 项目 | 参数 |
| --- | --- |
| 模式 | 上下火 |
| 温度 | 230℃ |
| 模具 | 6 寸，垫好油纸 |
| 时间 | 20-24 分钟 |

## 👨‍🍳 制作流程

1. **第一步：准备工作（提前做）**

    + 奶油奶酪提前 2 小时取出，软化到手指能轻松按出坑
    + 鸡蛋和淡奶油提前回温到常温
    + 烤箱提前预热 230℃
    + 6 寸模具垫好油纸（不用剪太整齐，褶皱随意）

2. **第二步：调抹茶糊（最关键）**

    + 抹茶粉过筛，加入 70℃ 左右热水，搅成顺滑无颗粒的膏状
    + 分 2-3 次加入淡奶油（约 40-50g），每次搅匀再加，直到变成顺滑的抹茶奶油液
    + 水温不要超过 80℃，否则颜色发黄、产生苦涩味

3. **第三步：制作奶酪糊**

    + 软化好的奶油奶酪用刮刀压拌到顺滑无颗粒
    + 细砂糖和玉米淀粉混合均匀后加入，继续搅拌到顺滑
    + 鸡蛋打散，分 3-4 次加入，每次搅拌均匀再加下一次
    + 鸡蛋必须常温：冷蛋液加入容易油水分离

4. **第四步：混合与过筛**

    + 抹茶奶油液倒入奶酪糊，搅拌均匀
    + 面糊过筛 1-2 次（不能省，成品更细腻）
    + 倒入模具，轻震两下震出大气泡

5. **第五步：烘烤**

    + 230℃ 烤 20-24 分钟
    + 抹茶上色不明显，以中心「duang duang」晃动为准；烤太久会失去流心
    + 表面上色太快可中途盖锡纸；抹茶通常不容易上色，一般不需要

6. **第六步：冷藏定型**

    + 出炉后彻底放凉（室温约 1 小时）
    + 连模具包好保鲜膜，冷藏至少 4 小时（最好过夜）

## 注意事项

+ 奶油奶酪软化：
    - 提前 2 小时取出，软化到手指能轻松按出坑
+ 抹茶粉结块：
    - 先用 70℃ 热水调成膏状，再和淡奶油混合；最后过筛可以挽救
+ 鸡蛋 / 淡奶油回温：
    - 必须常温，否则容易油水分离
+ 烤过头：
    - 抹茶上色不明显，以中心晃动为准判断熟度
+ 切片粘刀：
    - 每切一刀擦干净刀面，或用热刀切

## 📦 携带 / 打包

+ 提前一晚做好冷藏，出门前再拿出来
+ 脱模：拎着油纸提出，放在平整底托或盘子上
+ 装饰（可选）：表面筛一层薄抹茶粉，或放蜜红豆、挤奶油花
+ 打包：6 寸方盒，或保鲜膜包好放密封盒
+ 室温能撑 1-2 小时，短途一般没问题
