## 制作步骤（按时间线）

### 制作饼干底

奥利奥饼干+融化的黄油，搅拌均匀。

倒入模具底部，用力压紧压实（用杯底或勺子背压平），放入冰箱冷藏备用。

### 制作原味慕斯层

吉利丁片放入冰水中浸泡5-10分钟，泡软后取出**挤干水分**备用。

奶油奶酪室温软化后，加入细砂糖，用电动打蛋器低速搅**打至顺滑无颗粒**。

牛奶加热至微温（约50-60℃），放入泡软的吉利丁片，搅拌至完全融化。

将牛奶吉利丁液**分次倒入**奶酪糊中，边倒边搅拌，混合均匀。

淡奶油单独打发至**6分发**（提起打蛋头，奶油呈浓稠酸奶状，缓缓流动，出现纹路但很快消失）。

将奶酪糊倒入奶油中，用刮刀**翻拌均匀**（动作轻快，避免消泡）。

从冰箱取出模具，将原味慕斯糊倒入模具中，轻震两下震出气泡，放回冰箱**冷冻15-20分钟**（让第一层快速凝固，方便倒入第二层）。

### 制作抹茶慕斯层（趁第一层冷冻时）

抹茶粉加少量热水（约10-15g），用茶筅或小勺子调成无颗粒的糊状（不要有干粉疙瘩）。

重复“原味慕斯层”的操作：奶酪 + 糖搅打顺滑 → 牛奶+吉利丁融化 → 混合 → 加入抹茶糊搅拌均匀。

淡奶油单独打发至6分发，将抹茶奶酪糊倒入奶油中，翻拌均匀。

### 倒入第二层 & 冷藏定型

取出模具，确认原味层已表面凝固（手指轻按不粘手），将抹茶慕斯糊缓缓倒入第二层。

轻轻晃动模具使表面平整，轻震一下排出气泡。

放入冰箱冷藏4小时以上（最好过夜）。

### 脱模/装饰

冷藏定型后，用热毛巾敷在模具外壁30秒，或用电吹风热风沿模具边缘吹一圈。

将模具放在一个比底部小的瓶子上，轻轻向下推即可脱模。

食用前可在表面撒抹茶粉、放薄荷叶、挤奶油花，或放几颗蜜红豆

## 关键注意事项（重点看）

| 问题 | 原因 | 解决方法 |
| --- | --- | --- |
| 吉利丁泡发后直接扔进去？ | 必须挤干水分再放入温热液体，否则慕斯会出水、不成形 | 泡软后用手挤干，再放入50-60℃的牛奶中融化 |
| 奶酪糊有颗粒 | 奶油奶酪没软化到位 | 提前2小时从冰箱取出，或微波炉低火加热10秒再打 |
| 慕斯糊太稀，倒进去混层了 | 第一层没冻硬，或者慕斯糊温度太高 | 第一层一定要冷冻15-20分钟再倒第二层；慕斯糊要放凉至室温再倒入 |
| 抹茶层颜色发黄/有颗粒 | 抹茶粉没调开 | 抹茶粉先加少量热水调成细腻糊状，不要直接撒进奶酪糊 |
| 淡奶油打过头变成渣 | 打过了（超过6分发） | 淡奶油必须冷藏12小时以上再打，低速打发，看到出现纹路就停 |
| 成品口感硬/像果冻 | 吉利丁放多了 / 奶酪糊温度太低 | 严格按5g/片（1片）的量，不要多加 |
| 慕斯脱模后边缘不整齐 | 没热敷够 / 模具没清理干净 | 热毛巾多敷一会儿，用电吹风更均匀 |
