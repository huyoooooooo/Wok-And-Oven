# 布丁

奶香布丁，份量刚好约 2-3 杯（150ml 慕斯杯）。

## ✅ 配方（约 2-3 杯）

| 材料 | 克重 | 备注 |
| --- | --- | --- |
| 淡奶油 | 100g | — |
| 牛奶 | 100g | 全脂更香，低脂也行 |
| 细砂糖 | 15-20g | 15g 偏清淡，20g 正常甜度 |
| 蛋黄 | 3个（约 50g） | 常温；蛋清可留着炒菜 |
| 香草精 | 几滴（可选） | 没有就不加，不影响定型 |

## ⏰ 烤箱设置

提前 150℃ 预热

| 项目 | 参数 |
| --- | --- |
| 模式 | 上下火 |
| 温度 | 150℃ |
| 容器 | 150ml 慕斯杯 / 小烤碗，七八分满 |
| 时间 | 30-35 分钟（水浴；约 25 分钟起观察） |

## 👨‍🍳 制作流程

1. **第一步：加热奶液**

    + 淡奶油 + 牛奶 + 糖倒入小锅
    + 小火加热到边缘冒小泡、糖完全融化即可，不用煮开
    + 关火，放凉 2-3 分钟（太烫会把蛋黄烫成蛋花）

2. **第二步：混合蛋液**

    + 蛋黄打散，边搅拌边慢慢倒入温热奶液，搅匀
    + 有香草精就滴几滴搅匀

3. **第三步：过筛与装杯**

    + 过筛 1-2 次（布丁细腻的关键，不能省）
    + 倒入 150ml 慕斯杯或小烤碗，七八分满

4. **第四步：水浴烘烤**

    + 烤盘放布丁杯，倒入热水约 1-2cm（水浴法）
    + 150℃ 烤 30-35 分钟
    + 约 25 分钟起观察：表面凝固、中心微微晃动即可出炉

5. **第五步：冷藏**

    + 出炉放凉，冷藏 2 小时以上再吃

## 注意事项

+ 蛋液烫成蛋花：
    - 奶液加热后必须放凉 2-3 分钟，再慢慢倒入蛋黄，一边倒一边搅
+ 烤完有蜂窝孔洞：
    - 面糊一定要过筛；没过筛容易出气孔
+ 烤不熟 / 外熟内稀：
    - 用烤盘加热水做水浴，受热更均匀
+ 没有烤箱：
    - 可用蒸锅：水开后中火蒸 12-15 分钟，关火焖 5 分钟，冷藏后吃
