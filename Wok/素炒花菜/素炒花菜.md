# 素炒花菜（纯素版）

追求 干香、脆嫩、油亮

素炒花菜是最朴素的做法——调味简单，全程不加一滴水。和干锅花菜、手撕包菜的逻辑接近，都是「大火快炒、不加水、追求锅气」；区别在于花菜不需要焯水，掰好直接生炒，更简单。

## 📊 素炒花菜 vs 干锅花菜/番茄花菜

| 对比项 | 干锅花菜 | 番茄花菜 | 素炒花菜 |
| --- | --- | --- | --- |
| 预处理 | 焯水或微波断生 | 焯水断生 | 不需要焯水，直接生炒 |
| 配菜 | 五花肉+青红椒 | 番茄 | 大蒜+干辣椒（可选） |
| 调味 | 豆瓣酱+生抽蚝油 | 糖+盐+番茄酱 | 生抽+蚝油+糖+盐 |
| 汤汁 | 无，干香 | 有，酸甜汤汁 | 无，干香 |
| 特点 | 焦香入味 | 酸甜软韧 | 清甜脆嫩，干香适口 |

## 🥩 食材清单（2-3人份）

| 食材 | 用量 | 说明 |
| --- | --- | --- |
| 花菜 | 半个（约300克） | 掰成小朵，不要用刀切 |
| 大蒜 | 3-4瓣 | 切片或拍碎 |
| 干辣椒 | 2-3个（可选） | 剪段，不吃辣可省 |
| 食用油 | 2勺（30克） | 油量要够，花菜吃油 |
| 生抽 | 1.5勺（22克） | 咸味基底 |
| 蚝油 | 1勺（15克） | 提鲜 |
| 盐 | 最后尝味放 | 生抽蚝油有咸度，大概率不需要 |
| 白糖 | 小半勺（2克） | 提鲜 |

## 👨‍🍳 操作流程

1. 第一步：处理花菜（关键）

    + 花菜掰成小朵，不要用刀切
    + 用淡盐水泡10分钟（去虫卵和杂质），然后彻底沥干水分（带水下锅会出水，炒出来就不干香了）

2. 第二步：炒制

    + 锅烧热倒油，油量比平时炒菜略多（大火，油热到微微冒烟）
    + 下蒜片和干辣椒段爆香（中火，约20秒）
    + 转大火，下花菜快速翻炒（大火，约2-3分钟，炒到花菜表面微焦起皱、颜色变深）
    + 加生抽、蚝油、白糖调味（大火，快速翻炒均匀）
    + 尝味，淡了加少许盐（生抽蚝油有咸度，先尝再放）
    + 炒到花菜边缘微焦、油亮，出锅（大火，全程不加一滴水）

    全程大火，不加水——炒到花菜边缘微焦、颜色变深、表面油亮，就是熟了的标志。

## ✅ 成品标准

+ 口感
    脆嫩中带一点韧劲，不是软塌塌的
+ 盘底
    干爽，没有汤汁积水
+ 味道
    咸鲜微甜，花菜本身的清甜味还在
+ 观感
    花菜表面油亮，边缘有微微焦色

## 💡 带饭专项

+ 适合带饭吗？
    ✅ 可以，但回热后口感会从「脆」变成「韧」
+ 微波回热
    中高火1分30秒-2分钟
+ 保鲜期
    冷藏1-2天
