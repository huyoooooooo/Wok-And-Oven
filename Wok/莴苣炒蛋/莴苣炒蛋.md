# 莴苣炒蛋

属于“蔬菜+鸡蛋”的快手组合，但更清淡、更爽脆。核心：鸡蛋要嫩，莴苣要脆，全程大火快炒

## 🥩 食材清单（2-3人份）

| 食材 | 用量 | 说明 |
| --- | --- | --- |
| 莴苣 | 1根（约250-300克） | 去皮，切片或切丝 |
| 鸡蛋 | 2-3个 | 土鸡蛋更香 |
| 大蒜 | 2-3瓣 | 切末，爆香用 |
| 盐 | 约半小勺（2克） | 分两次——蛋液里放一点，炒莴苣时放一点 |
| 白胡椒粉 | 少许（1克） | 去蛋腥（可选） |
| 食用油 | 2勺（30克） | 炒蛋和炒莴苣各用一半 |

## 👨‍🍳 操作流程

1. **第一步：处理莴苣**

    + 莴苣削去外皮，削到露出翠绿色肉质（皮削干净，否则有粗纤维感）
    + 先切成长段，再切片或切丝，厚度约3-4mm（厚度均匀，熟度一致）
    + （可选）加少许盐腌制片刻，过水控干（能让莴苣更脆）

2. **第二步：炒鸡蛋**

    + 鸡蛋打散，加少许盐和白胡椒粉搅匀
    + 锅烧热倒油（约15克），油热后下蛋液（中大火，油温六成热）
    + 炒到蛋液刚刚凝固，用锅铲铲成块（中大火，约30-40秒，不要炒太老，嫩一点更好）
    + 盛出备用

3. **第三步：炒莴苣**

    + 锅中补油（约15克），下蒜末爆香（中火，约20秒）
    + 下莴苣片/丝，快速翻炒（大火，约1-2分钟，不要炒太久，保持脆度）
    + 加少许盐调味

4. **第四步：合炒出锅**

    + 倒回炒好的鸡蛋（大火）
    + 快速翻炒均匀（大火，约15-20秒）
    + 出锅装盘

## ✅ 成品标准

+ 莴苣：脆嫩爽口，颜色翠绿，没有软塌感
+ 鸡蛋：嫩滑金黄，和莴苣形成「脆+嫩」的口感对比
+ 整体：清淡咸鲜，清甜脆爽，适合配米饭或白粥

---

可加木耳：泡发几朵木耳，和莴苣一起炒，增加脆感层次，颜色也更好看
可加胡萝卜：胡萝卜切丝和莴苣一起炒，增加甜味和色彩
