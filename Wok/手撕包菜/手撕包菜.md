# 手撕包菜（纯素版）

追求 清爽干香

## 💡 核心逻辑

关键点

1. 手撕不要刀切	
    刀切会破坏包菜的纤维结构，手撕的断面不规则，更容易吸附味道，口感也更脆
2. 油要足	
    包菜很吃油，纯素版没有五花肉出油，油量一定要到位，否则炒出来干涩不香
3. 全程不加水	
    加水就变成“水煮包菜”了，干香口感彻底消失
4. 全程大火	
    锅要烧到足够热，大火快炒才能产生锅气
5. 包菜必须沥干	
    带水下锅=锅温骤降=出水成汤


## 🥩 食材清单（2餐带饭量）

| 食材 | 用量 | 说明 |
| --- | --- | --- |
| 包菜（圆白菜/卷心菜） | 半个（约300-400克） | 用手撕成小片，去硬梗 |
| 干辣椒 | 3-5个 | 剪段，不吃辣可省 |
| 大蒜 | 3-4瓣 | 切片 |
| 食用油 | 比平时炒菜多一倍 | 包菜吃油，纯素版油量必须足 |
| 生抽 | 1.5勺（22克） | 调味 |
| 蚝油 | 1勺（15克） | 提鲜 |
| 香醋 | 半勺（8克） | 出锅前沿锅边淋入，增香解腻 |
| 白糖 | 小半勺（2克） | 提鲜 |
| 盐 | 最后尝味放 | 生抽蚝油有咸度，大概率不需要 |

## 👨‍🍳 操作流程

1. 包菜手撕成小片（约4-5cm见方），去硬梗，洗净后彻底沥干水分（厨房纸吸干更好）
2. 锅烧热，倒油，油量比平时炒菜多一倍（大火，油热到微微冒烟）
3. 下蒜片和干辣椒段爆香（中火，约20秒，炒出香味，不要炒糊）
4. 转大火，下包菜（大火，锅要足够热）
5. 快速翻炒，让每一片包菜都裹上油（大火，约1.5-2分钟，炒到包菜变软、体积明显缩小）
6. 加生抽、蚝油、白糖调味（大火，快速翻炒均匀）
7. 沿锅边淋入香醋（大火，约20秒，醋沿锅边淋入，高温激发出醋香）
8. 尝味，淡了加少许盐（生抽蚝油有咸度，先尝再放）
9. 炒到包菜边缘微焦、油亮，出锅（大火，全程从下锅到出锅不超过3分钟）

## ✅ 成品标准

判断维度

+ 口感	
    爽脆中带一点韧劲，不是软塌塌的
+ 盘底	
    干爽，没有汤汁积水
+ 味道	
    咸鲜微辣，醋香若有若无，包菜本身的清甜味还在
+ 观感	
    包菜表面油亮，边缘有微微焦色

## ⚠️ 翻车预警

| 翻车现象 | 原因 | 改进 |
| --- | --- | --- |
| 包菜出水、盘底一摊水 | ①水分没沥干 ②火不够大 ③炒太久 ④加了水 | ①厨房纸吸干 ②全程最大火 ③从下锅到出锅不超过3分钟 ④绝对不加水 |
| 包菜软塌塌不脆 | 炒太久 | 看到包菜体积缩小、颜色变深就立刻调味出锅 |
| 不够香 | 油不够或火不够大 | 油量要足，锅要烧到冒青烟再下菜 |
| 醋味太冲 | 醋直接倒进菜里 | 醋要沿锅边淋入，高温激发出醋香再和菜混合 |
| 炒糊了 | 锅太干或油温过高 | 油量要够，包菜下锅后快速翻动 |

## 💡 带饭专项

+ 适合带饭吗？	
    ✅ 可以，但回热后口感会从“脆”变成“韧”，风味还在
+ 微波回热	
    中高火1分30秒-2分钟
+ 保鲜期	
    冷藏1-2天
