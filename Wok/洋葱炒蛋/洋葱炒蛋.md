# 洋葱炒蛋

洋葱自带甜味，出水慢，容错率高，是炒蛋系列里最适合新手练手的一道

## 🥩 食材清单（2餐带饭量）

| 食材 | 用量 | 说明 |
| --- | --- | --- |
| 洋葱 | 半个（约150克） | 紫皮为佳，辛辣味更足 |
| 鸡蛋 | 3个（大号） | 蛋液彻底打散至起泡 |
| 盐 | 1小勺（3克） | 分两次：蛋液2/3，洋葱1/3 |
| 白胡椒粉 | 少许（1克） | 去蛋腥 |
| 食用油 | 2勺（30克） | 比韭菜/韭黄略少 |

## 👨‍🍳 操作流程

1. 洋葱切丝，顺着纹路切，约 0.5-0.8cm 宽（不要切太细，否则炒烂了没口感）
2. 鸡蛋打散，加 2/3 的盐和白胡椒粉，彻底搅打至起泡（盐和胡椒粉在这一步就放好）
3. 锅烧热倒油，油面微微波动（中大火），下蛋液（不要立刻翻动，等底部凝固）
4. 推炒至七成熟（表面还带湿润），中大火约 30-40 秒，鸡蛋炒到蓬松就盛出备用
5. 锅里留底油（不够补一点），转中大火，下洋葱丝，约 1.5-2 分钟，炒到变软、变透明，边缘微微焦黄
6. 加剩余 1/3 的盐，中大火继续翻炒至洋葱完全透明
7. 转中小火，倒回鸡蛋，轻轻翻炒 10-15 秒，让鸡蛋和洋葱混合均匀，出锅

## ✅ 关键判断节点

1. 洋葱什么状态加鸡蛋	
    洋葱丝全部变透明、边缘微焦，锅底没有积水
2. 倒回鸡蛋后怎么办	转中小火，轻轻翻炒10-15秒，让两者融合

## 💡 带饭专项

+ 适合带饭吗？	
    ✅ 非常适合
+ 装盒	
    直接装，汤汁极少，干爽
+ 微波回热	
    中高火1分30秒-2分钟
+ 保鲜期	
    冷藏2-3天
