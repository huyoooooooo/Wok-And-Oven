# 莴苣炒肉

和莴苣炒蛋，套路很像，核心都是保持莴苣的脆嫩，配菜从鸡蛋换成了肉片

## 核心操作：肉片嫩滑 + 莴苣脆爽

同时照顾肉和菜两个“脾气不同”的食材：

+ 肉片	嫩滑不柴	腌制上浆（淀粉锁水），滑炒变色即出
+ 莴苣	脆嫩不出水	大火快炒，断生即出锅，不能久煮

双椒鸡丁类似——肉先炒熟盛出，莴苣炒断生，最后回锅合炒，全程不超过几分钟。

## 🥩 食材清单（2-3人份）

| 食材 | 用量 | 说明 |
| --- | --- | --- |
| 猪瘦肉（里脊/腿肉） | 150-200克 | 切片，厚度约2-3mm |
| 莴苣 | 1根（约250-300克） | 去皮切片 |
| 大蒜 | 2-3瓣 | 切末 |
| 姜 | 1小块 | 切末 |

**腌肉料**

| 调料 | 用量 |
| --- | --- |
| 料酒 | 1勺（15克） |
| 生抽 | 1勺（15克） |
| 白胡椒粉 | 少许（1克） |
| 玉米淀粉 | 1勺（15克） |
| 食用油 | 1勺（15克） |

**炒制调料**

| 调料 | 用量 |
| --- | --- |
| 生抽 | 1勺（15克） |
| 蚝油 | 半勺（8克） |

| 食用油 | 约2勺（30克） |
## 👨‍🍳 操作流程

1. **第一步：处理肉片（参考你之前做彩椒牛柳的经验）**

    + 猪肉切薄片（约2-3mm），顺着纹路切。
    + 加料酒、生抽、白胡椒粉抓匀，加淀粉抓匀上浆，最后加食用油封住。
    + 腌制10-15分钟。

2. **第二步：处理莴苣（参考莴苣炒蛋）**

    + 莴苣削去外皮（削到露出翠绿色肉质），切片，厚度约3-4mm。
    + （可选）撒少许盐腌5-10分钟，杀出水分后冲洗沥干——这样炒出来更脆。

3. **第三步：滑炒肉片（和双椒鸡丁一样，变色即出）**

    + 锅烧热倒油，油热后下腌好的肉片（中大火，油温六成热）
    + 快速划散，炒到肉片表面变色（中大火，约1-2分钟，变色即盛出，不要炒全熟）
    + 盛出备用

4. **第四步：炒莴苣**

    + 锅中留底油，下姜蒜末爆香（中火，约20秒）
    + 下莴苣片，快速翻炒（大火，约1-2分钟，炒到莴苣断生、颜色变深绿）

5. **第五步：合炒出锅**

    + 倒回肉片，加生抽、蚝油（大火）
    + 快速翻炒均匀（大火，约20-30秒，炒匀即可，不要恋战）
    + 尝味，淡了加盐
    + 出锅## ✅ 成品标准

## ✅ 成品标准

+ 肉片：嫩滑不柴，表面有光泽
+ 莴苣：脆嫩爽口，颜色翠绿，没有软塌感
+ 整体：咸鲜清爽，肉香和莴苣的清甜融合
