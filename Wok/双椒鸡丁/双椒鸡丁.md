# 双椒鸡丁

和彩椒牛柳是同一套节奏：腌肉→滑炒→炒配菜→回锅裹汁。

## 🥩 食材清单（2-3人份）

| 食材 | 用量 | 说明 |
| --- | --- | --- |
| 鸡腿（去骨） | 2-3个（约300-400克） | 去皮不去皮都可以，切1.5cm丁 |
| 青椒 | 2个 | 切丁，和鸡丁大小接近 |
| 红椒 | 1-2个 | 切丁，配色用 |
| 大蒜 | 3-4瓣 | 切末 |
| 姜 | 1小块 | 切末 |
| 食用油 | 比平时炒菜略多 | 滑炒用 |

**腌鸡丁料**

| 调料 | 用量 |
| --- | --- |
| 料酒 | 1勺（15克） |
| 生抽 | 1勺（15克） |
| 白胡椒粉 | 少许（1克） |
| 玉米淀粉 | 1勺（15克） |
| 食用油 | 1勺（15克，封油） |

**调味汁（提前调好）**

| 调料 | 用量 |
| --- | --- |
| 生抽 | 1.5勺（22克） |
| 蚝油 | 1勺（15克） |
| 白糖 | 小半勺（2克） |
| 白胡椒粉 | 少许（1克） |
| 水淀粉 | 2勺（约30ml水+半勺淀粉） |
## 👨‍🍳 操作流程

1. **第一步：处理鸡肉（和彩椒牛柳一样）**

    + 鸡腿去骨，切成1.5cm见方的小丁。
    + 加料酒、生抽、白胡椒粉、淀粉抓匀，腌制15-20分钟。
    + 最后加1勺食用油封住，防粘。

2. **第二步：滑炒鸡丁**

    1. 锅烧热倒油，油热后下鸡丁（中大火，油温六成热）
    2. 快速划散，炒到鸡丁表面变色、微黄（中大火，约1-2分钟，变色即出，不要炒全熟）
    3. 盛出备用

3. **第三步：炒双椒**

    1. 锅中留底油，下姜蒜末爆香（中火，约20秒）
    2. 下青红椒丁翻炒（中火，约1-2分钟，炒到辣椒断生、颜色变深，保持脆度）

4. **第四步：合炒收汁**

    1. 倒回鸡丁，快速翻炒均匀（大火，约30秒）
    2. 淋入调好的汁（生抽+蚝油+糖+水淀粉），快速翻炒（大火，约30秒，让酱汁裹住每一块鸡丁）
    3. 汤汁变浓稠后出锅（留一点汁拌饭）

## ✅ 成品标准

+ 鸡丁：嫩滑不柴，表面裹着薄薄的酱汁
+ 青红椒：脆嫩，颜色鲜艳，断生但不过软
+ 酱汁：浓稠适量，盘底有余汁

适合带饭。鸡丁回热后口感变化不大，汤汁会进一步渗入鸡肉。微波中高火2分钟即可。
