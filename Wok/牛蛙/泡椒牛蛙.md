# 泡椒牛蛙。

“酸辣牛蛙”里最经典，是川渝家常菜里的下饭担当。追求牛蛙的“嫩滑”，风味靠的是泡菜发酵出来的酸辣醇香，更浓郁、更开胃。

和双椒鸡丁类似，核心都是“先滑油定型，再合炒入味”，全程大火快节奏，最后带汤汁出锅。

## 🥩 食材清单（2-3人份）

| 食材 | 用量 | 说明 |
| --- | --- | --- |
| 牛蛙 | 2-3只（约500-750克） | 现杀最好，斩成拇指大小的块 |
| 泡红辣椒 | 100-150克 | 切段或切碎，酸辣味主力 |
| 泡野山椒 | 50克（可选） | 增加酸辣层次 |
| 泡姜 | 20克 | 切片或切末 |
| 大蒜 | 3-4瓣 | 切片或拍碎 |
| 大葱 | 1根 | 切段 |
| 配菜（可选） | 适量 | 莴笋、黄瓜、丝瓜等，切条或切块 |

**腌牛蛙料**

| 调料 | 用量 |
| --- | --- |
| 料酒/啤酒 | 1-2勺（15-30克） |
| 盐 | 半小勺（2克） |
| 白胡椒粉 | 少许（1克） |
| 玉米淀粉 | 1勺（15克） |

**炒制调料**

| 调料 | 用量 |
| --- | --- |
| 食用油 | 比平时炒菜略多 |
| 生抽 | 1勺（15克） |
| 白糖 | 小半勺（2克） |
| 泡椒水 | 2-3勺（30-45克） |
| 盐 | 最后尝味放 |
## 👨‍🍳 操作流程

1. **第一步：处理牛蛙（和鲜椒牛蛙一样）**

    + 牛蛙洗净斩块，大小均匀（约拇指大小，大小一致，熟度才均匀）
    + 加料酒、盐、白胡椒粉抓匀，腌制15分钟（牛蛙会出水）
    + 倒掉腌出的血水（这步不能省，否则淀粉挂不住）
    + 加淀粉抓匀上浆（薄薄一层即可）

2. **第二步：滑油定型（和双椒鸡丁一样，变色即出）**

    + 锅烧热倒油，油温五六成热（筷子插入冒泡）（中火，油温不能太高，否则肉会老）
    + 下腌好的牛蛙块，快速划散（中火，约1分钟，炒到牛蛙变色、七成熟即盛出）
    + 盛出备用（不要炒全熟，后面还要焖煮）

3. **第三步：炒底料（中小火，泡椒的酸香味要逼出来）**

    + 锅中留底油，下泡姜、大蒜爆香（中小火，约20秒，蒜和姜先出香）
    + 下泡红辣椒、泡野山椒（切碎后味道更浓）（中小火，约1-1.5分钟，炒到泡椒的酸香味出来，红油透亮）
    + （可选）加半勺郫县豆瓣酱炒出红油（中小火，约30秒，增色增香，注意不要炒糊）

4. **第四步：合炒焖煮（全程大火，3-5分钟出锅）**

    + 倒回牛蛙，加生抽、白糖、泡椒水翻炒均匀（大火，约30秒，泡椒水是酸辣味的灵魂）
    + 加小半碗水或高汤（没过食材一半），盖上锅盖焖煮（中火，约3-5分钟，让牛蛙吸味，配菜如莴笋可在这步一起下）
    + 尝味，淡了加盐（泡椒和豆瓣酱都有咸度，盐最后放）
    + 不要收太干，留一些汤汁（汤汁拌饭是这道菜的精华）
    + 出锅，撒葱花

总用时：约25-30分钟。

## 成品标准

+ 牛蛙	嫩滑但筷夹不散，肉质雪白，咬开有弹性
+ 汤汁	酸辣醇厚，泡椒的香气融入汤汁和肉里，留有余汁拌饭
+ 颜色	红亮油润，泡椒的红色和豆瓣酱的红油交融
+ 风味	酸辣开胃，泡椒的发酵酸香浓郁，吃完一碗饭还想再添一碗
