# 鲜椒牛蛙

以“牛蛙嫩滑”为核心，烹饪方法和双椒鸡丁类似：先滑油定型，再入汤烧入味。

鲜椒牛蛙的“鲜”，核心来自鲜花椒和新鲜辣椒的搭配，一种清冽的麻香配上鲜辣的椒香。

## 选料与处理

牛蛙的处理方法和泡椒牛蛙完全一致——腌制后一定要把腌出的血水倒掉，否则后淀粉挂不住，肉会腥。

+ 牛蛙让摊主杀好去皮，回家洗净斩块，厚肉处可划一刀方便入味（大小均匀，约3cm块）
+ 加料酒、盐、白胡椒粉抓匀，腌制15分钟（牛蛙会出水）
+ 倒掉腌出的血水（这一步不能省，水分不去掉，淀粉挂不住）
+ 加红薯淀粉或玉米淀粉抓匀上浆（薄薄一层即可）

## 🥩 配料与调味（鲜椒风味的核心）

| 食材 | 用量 | 说明 |
| --- | --- | --- |
| 青小米辣 | 100克 | 切碎，辣味主力 |
| 红小米辣 | 100克 | 切碎，增色增辣 |
| 仔姜/老姜 | 各适量 | 仔姜切丝或粒，风味更清新 |
| 大蒜 | 50克 | 拍散剁碎 |
| 鲜花椒 | 1把 | 灵魂配料，清冽麻香来源 |
| 酸菜杆 | 适量（可选） | 切碎，增加复合酸香 |
| 豆瓣酱 | 少量 | 炒红油增色，不要多放，否则抢味 |
| 食用油 | 约50克 | 炒底料用 |

## 👨‍🍳 操作流程（全程约25分钟）

1. **第一步：炒底料、熬鲜椒汤底（先熬汤，再下蛙）**

    特点是先熬出麻辣鲜香的汤底，再把牛蛙放进去烫熟，牛蛙在锅里只煮5分钟左右，所以特别嫩。

    + 锅烧热倒油（约50克），下青花椒和酸菜末炒香（中火，约20秒，酸菜可选，增加酸香层次）
    + 下少许豆瓣酱炒出红油（中火，约30秒，豆瓣酱不要多，主要是提色）
    + 下大蒜末、姜末一起爆香（中火，约20秒）
    + 加入适量清水，放入鲜花椒，大火烧开（大火）
    + 加盐（约7克）、胡椒粉（约1克）、少许白糖、生抽（约5克）（汤汁要稍微偏咸，蛙肉才能入味）
    + 转小火，熬煮5分钟（让花椒和辣椒的香味充分释放）
    + 下青红小米辣碎和仔姜丝

2. **第二步：下牛蛙（全程5分钟，时间到了就出锅）**

    + 下腌制好的牛蛙，轻轻推散（小火，5分钟，不要大火翻滚，蛙肉容易散）
    + 中途加入剩余的姜蒜末（让味道更浓郁）
    + 时间到后加鸡精/味精调味
    + 牛蛙夹出摆盘，浇入原汤

3. **第三步：泼油增香（可选，风味提升的关键）**

    + 锅中烧油至七成热（约200℃）
    + 碗中放鲜花椒，热油泼上去，激发出麻香（这一步能让麻味更冲、更清新）
    + 将花椒油淋在牛蛙上即可

    泼油步骤非必须的，但此操作使得鲜花椒清冽的麻香味更明显，和「椒麻鸡丝凉面」里泼花椒油的思路是一致的。

## 成品标准

+ 牛蛙	嫩滑但不散，筷子夹起来不碎，肉质有弹性
+ 汤汁	清亮微红，麻辣味突出，有鲜椒的清鲜感
+ 风味	麻辣鲜香，辣得清爽，不燥不腻

---

和泡椒牛蛙的操作思路一样——都是先熬汤底、再下牛蛙煮5分钟

区别在于鲜椒用的是鲜椒靠的是新鲜辣椒和鲜花椒的清鲜麻辣，辣味更直接、更清爽，吃完不会觉得厚重；泡椒用的是泡野山椒和泡姜，味道更酸辣浓郁。
