# 白菜烧牛肉（熟牛肉版）

熟牛肉的好处是省去了炖煮时间，不用等它变软；缺点是牛肉本身已经熟了，不容易再“吸味”

所以核心不是“把牛肉炖烂”，而是用汤汁把牛肉回热并裹上味道，同时让白菜吸收牛肉和汤汁的香味。

## 🥩 食材清单（2-3人份）

| 食材 | 用量 | 说明 |
| --- | --- | --- |
| 熟牛肉（酱牛肉/卤牛肉） | 200-250克 | 切片或切条，厚度约3-4mm |
| 大白菜 | 4-5片（约300-400克） | 帮叶分开，帮子斜刀片，叶子撕大块 |
| 干辣椒 | 2-3个 | 剪段，不吃辣可省 |
| 大蒜 | 3-4瓣 | 切片或拍碎 |
| 食用油 | 2勺（30克） | 炒制用 |

**调味汁（提前调好）**

| 调料 | 勺 | 克重 |
| --- | --- | --- |
| 生抽 | 2勺 | 30克 |
| 老抽 | 半勺 | 8克 |
| 蚝油 | 1勺 | 15克 |
| 白糖 | 半小勺 | 2克 |
| 白胡椒粉 | 少许 | 1克 |
| 清水 | 约半碗 | 80-100ml |

## 👨‍🍳 操作流程

1. 第一步：处理白菜（和白菜烧豆腐一样）

    + 白菜帮子和叶子分开
    + 帮子部分斜刀片成薄片
    + 叶子部分撕成大块

2. 第二步：切牛肉

    熟牛肉切成薄片或条，厚度约3-4mm

    如果牛肉是从冰箱拿出来的，切之前回温一下，或者切薄一点，方便回热

3. 第三步：煸炒

    + 锅烧热倒油，下蒜片和干辣椒爆香（中火，约20秒）
    + 下白菜帮子，中大火翻炒到边缘变透明（中大火，约1.5-2分钟）
    + 下白菜叶子，翻炒到变软（中大火，约1分钟）

4. 第四步：合烧

    + 下切好的熟牛肉，轻轻翻炒均匀（牛肉不要炒太久，容易碎）
    + 淋入调好的调味汁（生抽+老抽+蚝油+糖+胡椒粉+清水）
    + 盖上锅盖，中小火焖3-5分钟（让牛肉回热，白菜吸味）
    + 开盖，尝味（淡了加少许盐）
    + 转大火收汁，留一些汤汁（大火，约1分钟，汤汁收浓即可，不要收干）

## ✅ 成品标准

+ 牛肉  软而不烂，表面裹着酱汁
+ 白菜  软甜吸汁，边缘微透明
+ 汤汁  浓稠适中，盘底有余汁
+ 风味  酱香咸鲜，牛肉味和白菜清甜融合

## 🥩 关于“熟牛肉”的选择
+ 酱牛肉/卤牛肉（✅ 首选）
    本身有咸味，切薄片后烧一下，风味融合
+ 红烧牛肉（剩菜）（✅ 可以）
    直接切块或切片，连汤汁一起用
+ 白切牛肉（✅ 也可以）
    本身没味道，需要靠调味汁补足
+ 冷冻熟牛肉（⚠️ 解冻后切薄片）
    和鲜熟牛肉处理方式一样

---

白菜属于叶子菜，是需要焖煮个一会儿吸润汤汁的味道，生牛肉不适合，反而熟牛肉更好
