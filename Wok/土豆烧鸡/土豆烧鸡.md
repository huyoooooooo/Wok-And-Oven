# 土豆红烧鸡腿

算是葱烧鸡腿的另一个方向——不加香料，加辣(可选)，用酱油、糖和土豆的甜，焖出一锅浓稠红亮的家常味。

## 🥩 食材清单（2-3人份，2餐饭量）

| 食材 | 用量 | 说明 |
| --- | --- | --- |
| 鸡腿（琵琶腿） | 3-4个（约500克） | 剁块 |
| 土豆 | 2个（约300克） | 滚刀块 |
| 大葱 | 1根 | 切段，调味用 |
| 姜 | 4-5片 | — |
| 蒜 | 3-4瓣 | 拍一下 |
| 生抽 | 2勺（30克） | 咸味基底 |
| 老抽 | 1勺（15克） | 上色，比葱烧版多一点点，颜色更红亮 |
| 蚝油 | 1勺（15克） | 提鲜 |
| 冰糖 | 5-6颗（约10克） | 提亮汤汁 |
| 料酒 | 1勺（15克） | 去腥 |
| 盐 | 最后尝味放 | 生抽蚝油有咸度，大概率不用加 |
| 食用油 | 1勺（15克） | 煎鸡腿用 |

## 👨‍🍳 操作流程

1. 鸡腿剁块，新鲜直接煎，冷冻先焯水
2. 土豆切滚刀块，泡在清水里防止氧化（这一步提前备好）
3. 锅烧热倒油，鸡皮朝下铺入锅中，中小火煎约 2-3 分钟至金黄焦脆（鸡皮一定要煎透）
4. 翻面，中小火约 1-2 分钟煎到微黄
5. 下大葱段、姜片、蒜瓣，中火约 1 分钟炒出香味
6. 加生抽、老抽、蚝油、冰糖、料酒，翻炒让鸡块裹上酱色（老抽比葱烧版多半勺，颜色更红亮）
7. 加开水，水量没过鸡块
8. 盖盖，中小火焖煮 10 分钟，先让鸡肉入味
9. 下土豆块（沥干水分），轻轻推匀（土豆不要大力翻动，容易碎）
10. 继续盖盖焖煮 10-12 分钟，炖到土豆筷子能戳透
11. 开盖，尝味，淡了加盐
12. 转大火收汁至汤汁浓稠（留一些汤汁拌饭）
13. 出锅

## ✅ 成品标准

+ 鸡腿	
    皮微焦，肉嫩入味，筷子能轻松戳动
+ 土豆	
    软烂但保持形状，边缘微微炖化进汤汁里，筷子一戳就透
+ 汤汁	
    红亮浓稠，能挂在鸡块和土豆上，盘底有余汁
+ 风味	
    酱香+微甜（来自冰糖和土豆的甜），没有香料味，纯粹家常
