# 烧椒豆花牛肉

这是一道经典的川渝家常菜，灵魂在于“烧椒”——青椒经过干烧或明火烤制产生的焦香，混合牛肉的嫩滑和豆花的绵软。

按照“烧椒→牛肉→合烧” 顺序来

## 食材处理

### 烧椒的做法
1. 青椒洗净，不去蒂(去蒂切段更方便)，用厨房纸擦干表面水分(有条件，无条件沥干)
2. 铁锅烧热，不放油，中小火，把青椒直接放进去
3. 用铲子不断按压、翻面，让青椒表面均匀受热
4. 煎到表皮大面积起皱、发黑、出现焦糊斑，内部变软(2,3,4干煸)
5. 取出放入碗中，盖上盖子或保鲜膜焖2-3分钟
6. 稍微放凉后，撕掉焦黑的外皮（可保留部分焦皮增加风味）
7. 去蒂 去籽(看个人)，剁成粗末（不要太碎，保留颗粒感）

💡 正宗的烧椒，就是用这种“火烧”或“干烧”的方式，目的是让辣椒产生独特的焦香味，这是这道菜的味觉灵魂。

### 牛肉腌制（嫩滑关键）
1. 牛里脊逆着纹理切成薄片（厚度2-3mm）
2. 加入腌料：生抽1勺、料酒1勺、白胡椒粉少许、盐少许
3. 加入玉米淀粉2小勺，抓拌均匀，让每片牛肉都裹上薄粉
4. 最后淋入食用油1小勺，抓匀锁水

腌制15分钟

### 豆花预处理

+ 嫩豆腐
烧一锅水，水开后加少许盐，放入掰开的豆腐块焯水1分钟，捞出沥干，铺在大碗底部
+ 豆花/内脂豆腐
一般都是直接可食用，直接铺在碗底即可

## 菜谱制作

+ 步骤1：炒烧椒酱
    - 热锅倒油（比平时炒菜稍多），放入姜末、蒜末，小火爆香
    - 加入剁碎的烧椒，中火翻炒1-2分钟，炒出焦香，可加入豆豉1小勺一起炒

+ 步骤2：煮汤汁
    - 倒入清水约400ml，大火烧开
    - 尝味，不够咸补盐，加入生抽、耗油、白糖提鲜(看个人)

+ 步骤3：滑牛肉
    - 转小火，保持汤汁微沸状态（不要大滚），将腌好的牛肉片一片一片铺入汤汁中
    - 全部下锅后，用铲子轻轻推动，让牛肉均匀受热
    - 煮到牛肉变色、完全熟透即可关火（约1-2分钟），不要久煮

+ 步骤4：合体出锅
    - 将煮好的牛肉和汤汁一起，浇在铺好豆花的大碗中

表面撒葱花、花椒粉（或淋几滴花椒油），喜欢吃辣的可以再淋一勺辣椒油。

## 说明
如果有喜欢吃辣的伙伴，可以在碗面，放上花椒、干辣椒、浇上一勺热油计划香味
