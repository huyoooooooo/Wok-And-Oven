# 烧椒豆花牛肉

这道菜的三个核心元素

| 元素 | 角色 | 核心要求 |
| --- | --- | --- |
| 烧椒 | 风味灵魂 | 焦香突出，不能有生辣味，也不能烧糊发苦 |
| 豆花 | 口感基底 | 嫩滑如凝脂，入口即化，但不能散碎 |
| 牛肉 | 鲜味主角 | 滑嫩不柴，裹满烧椒汤汁 |

风味逻辑：烧椒的焦香 + 牛肉的鲜 + 豆花的清甜 = 咸鲜微辣、层次丰富。
家常做法：先滑炒后炖煮，成品带汤汁

## 🥩 食材量化（2餐带饭量）

| 食材 | 用量 | 说明 |
| --- | --- | --- |
| 牛里脊（或吊龙） | 250克 | 逆纹切薄片，带雪花的最好 |
| 内酯豆腐/嫩豆花 | 1盒（约300克） | 代替豆花，更易买，口感接近 |
| 二荆条（青线椒） | 4-5根 | 做烧椒用，不辣但清香 |
| 大蒜 | 4-5瓣 | 部分做烧椒，部分爆香 |
| 洋葱 | 小半个 | 垫底用（可选，增香） |

| 调料 | 用量 | 说明 |

| 盐 | 半小勺 | 最后尝味放 |

## 👨‍🍳 分步拆解（重点关注烧椒制作+牛肉嫩滑）

1. 烧椒制作（这道菜的"灵魂"）
    - 二荆条洗净，不去蒂（好拿）
    - 锅烧热，不放油，下青椒（中小火，干煸）
    - 用铲子不停按压，让辣椒表面均匀起皱、出现焦黑虎皮斑（约3-5分钟，不要烧成黑炭）
    - 盛出放凉，撕掉表面的焦黑外皮（不撕也可以，但撕了口感更好）
    - 剁碎，和蒜末混合（这就是烧椒碎）

关键判断：烧椒的香味是焦香，不是苦味。如果表皮全黑了、甚至冒烟了，就是烧过头了，会发苦。看到表面起皱带虎皮斑、辣椒变软就立刻盛出。

2. 牛肉腌制（嫩滑的关键）
    - 牛里脊逆纹切薄片（约2-3mm厚）
    - 加生抽、蚝油、白胡椒粉、淀粉抓匀
    - 分次加少量水抓匀（约2-3勺）（让牛肉"吃水"，炒出来更嫩）
    - 加食用油封油（锁住水分，防粘）
    - 腌制10-15分钟

如果想让牛肉更嫩：可以加1/4小勺小苏打和腌料一起抓匀，腌10分钟后冲洗一下。

3. 烹饪流程（做法A：滑炒后炖煮，做法B:直接在烧椒汤汁中滑熟）
    - 豆花（内酯豆腐）加盐开水焯水1分钟，捞出铺在碗底（中火，1分钟，加盐给底味，焯水让豆腐更紧实）
    - 热锅凉油，下腌好的牛肉片，快速滑炒至变色断生（大火，约30秒，变色即盛出，不要炒全熟）
    - 锅留底油，下蒜末和烧椒碎爆香（中火，约1分钟，炒出烧椒的焦香味）
    - 加生抽、蚝油、白糖，倒入清水约300ml-400ml煮开（中火，调成烧椒汤汁）
    - 倒回牛肉片，煮至牛肉全熟（中小火，约1分钟）
    - 将烧椒牛肉连汤汁一起浇在豆花上（汤汁没过豆花一半以上）
    - 撒葱花，完成

## 💡 带饭专项

适合带饭：非常适合。汤汁丰富，微波回热后豆花和牛肉更入味

装盒建议：豆花铺底部，牛肉和烧椒汤汁浇在上面。汤汁要多留一些

微波回热：中高火2分30秒-3分钟，中途搅拌一次

绝配主食：白米饭，汤汁拌饭是灵魂
