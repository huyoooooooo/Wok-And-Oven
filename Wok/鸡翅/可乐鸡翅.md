# 可乐鸡翅

风味核心是“甜咸平衡、酱汁浓稠、色泽红亮”，做法更像葱烧鸡腿/土豆红烧鸡腿，是“煎香→加液体焖煮→收汁”的路子，比干锅鸡翅省事，比蒜香鸡翅更甜润。

是厨房新手第一次做就成功的菜，容错率很高——因为可乐自带糖分和焦糖色，能自然地让鸡翅上色、让汤汁变浓，不需要你操心“老抽放多少才够红”或者“糖放多了会不会甜”这种问题。

## 💡 味型逻辑

这道菜虽然叫“可乐鸡翅”，但它不是甜腻的——成品是甜咸适中、带有可乐特有的焦糖香气和微微的酸香，味道层次比“糖+酱油”更丰富。

关于可乐的选择：要用普通含糖可乐，不要用无糖或代糖的版本——因为无糖可乐加热后容易产生苦味，和鸡翅一起焖煮会让成品风味打折扣。

鸡翅选什么：这道菜最推荐用鸡翅中，大小均匀、肉质嫩滑、容易入味，成品也最漂亮。当然用全翅或者翅根也完全没问题，效果差不多。

## 🥩 食材清单（2-3人份）

| 食材 | 用量 | 说明 |
| --- | --- | --- |
| 鸡翅中 | 10-12个（约500克） | 翅中最佳，翅根全翅也可，洗净后厨房纸吸干 |
| 可乐 | 1罐（330ml） | 普通含糖可乐，不要用无糖的 |
| 生抽 | 2勺（30克） | 提鲜上色 |
| 老抽 | 半勺（8克） | 加深颜色 |
| 料酒 | 1勺（15克） | 焯水或腌制用 |
| 姜 | 3-4片 | 去腥 |
| 葱 | 1根 | 切段 |
| 八角 | 1个（可选） | 增加复合香味，不加也可以 |
| 食用油 | 少许 | 煎鸡翅用 |

## 👨‍🍳 操作流程

1. **第一步：处理鸡翅**

    + 鸡翅洗净，两面各划两刀（方便入味）。
    + 冷水下锅，加姜片和1勺料酒，大火煮开后焯水2-3分钟，撇去浮沫，捞出用温水冲净，沥干水分。

    + **为什么要焯水：** 这个做法和椒盐鸡翅不一样——椒盐鸡翅直接煎就能把水分逼掉，但可乐鸡翅需要先焯水去血沫，成品汤汁才干净清亮。有的做法会省掉焯水直接煎，但出来的汤汁容易浑浊。

2. **第二步：煎鸡翅**

    1. 平底锅烧热，倒少许油（中火）— 油量不用多，润锅即可
    2. 鸡翅正面朝下平铺入锅（中火，煎2-3分钟）— 煎到一面金黄再翻面
    3. 翻面煎另一面（中火，煎2-3分钟）— 两面都煎到金黄即可

    + **这一步的作用：** 把鸡皮里的油脂煎出来，同时让表皮产生焦香，是后面入味的基础。

3. **第三步：焖煮**

    1. 煎好的鸡翅留在锅里，加入姜片、葱段（八角可选）
    2. 倒入可乐，生抽2勺，老抽半勺 — 可乐量以基本没过鸡翅为准，330ml一罐刚好
    3. 大火烧开，转中小火焖煮（中小火，约15-20分钟）— 盖盖焖煮，中途可以翻动1-2次

4. **第四步：收汁（最关键的一步）**

    1. 开盖，转大火收汁（大火）— 收汁时要用铲子不停翻动鸡翅，防止糊锅
    2. 汤汁变浓稠、能裹住鸡翅表面时 — 颜色应该是红亮的焦糖色，锅中渗出一层明油
    3. 出锅装盘，撒熟白芝麻或葱花点缀

    + **收汁的判断标准：** 收汁要收到汤汁浓稠、鸡翅表面裹上焦糖色、锅中有明油渗出时出锅。可乐里的糖在高温下会浓缩成稠厚的酱汁，翻动不及时就容易糊，这一步要留意。

## ✅ 成品标准

+ 颜色：红亮油润的焦糖色，均匀裹在鸡翅表面
+ 口感：表皮微韧，内部软嫩多汁，轻松脱骨
+ 味道：甜咸平衡，有可乐特有的焦糖香和微酸尾调
+ 汤汁：浓稠挂勺，裹住鸡翅，盘底有余汁

## 💡 带饭专项

+ 适合带饭吗？	
    ✅ 非常适合。酱汁裹住鸡翅，回热后风味不减，反而更入味
+ 微波回热	
    中高火2分钟即可
+ 保鲜期	
    冷藏2天

这道菜对新手非常友好——无需操心调味平衡，可乐里的糖分和焦糖色会辅助上色。只需留意收汁时要保持翻动，别糊了。
