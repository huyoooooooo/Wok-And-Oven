# 干锅鸡翅

和椒盐鸡翅、蒜香鸡翅，是三种完全不同的路子：椒盐鸡翅是干香，蒜香鸡翅是软嫩带汁，而干锅鸡翅——是麻辣鲜香、外焦里嫩。

核心逻辑，和干锅花菜、手撕包菜是一脉相承的：追求干香，不加一滴水，靠底料爆香，用大火快炒把味道“逼”进食材里。但鸡翅比花菜“难搞”多了——它肉厚，得先把它弄熟、弄出焦香，再下锅和调料一起“干炒”。

## 💡 在家做，怎么省事又好吃？

对“鸡翅怎么熟”有两种主流思路：

1. 油炸版	
    + 鸡翅裹粉，用足量油炸到金黄酥脆	
    + 外皮最酥脆，干香感最强

2. 香煎版	
    + 平底锅少油，中小火慢煎到两面金黄焦脆	
    + 同样能出焦香，用油少，操作更友好	

不想油炸的话，用平底锅“煎”是完全可以替代“炸”的（和做椒盐鸡翅时的煎法一样）。煎的时候用中小火耐心一点，把鸡翅本身的油脂逼出来，一样能达到“表皮焦香”的效果。

## 🥩 食材清单（2-3人份）

**主料**

| 食材 | 用量 | 说明 |
| --- | --- | --- |
| 鸡翅中 | 8-10个（约500克） | 个头均匀，洗净后用厨房纸吸干水分 |

**配菜（干锅的"标配"，可根据喜好调整）**

| 食材 | 用量 | 说明 |
| --- | --- | --- |
| 土豆 | 1个（约150克） | 切粗条或厚片，煎到焦香，非常吸味 |
| 莲藕 | 半节（约100克） | 切片，焯水或直接煎，口感脆爽 |
| 洋葱 | 半个 | 切块，打底增香 |
| 青椒/红椒 | 各半个 | 切块，配色和增加清香味 |

配菜的选择很灵活，像芹菜、花菜、腐竹、年糕也都是干锅里的常客。不过第一次做，建议先试试经典搭配：土豆+莲藕+洋葱+青红椒。

**腌鸡翅料**（和椒盐/蒜香鸡翅的腌料类似，但这是干锅的底味）

| 调料 | 用量 | 说明 |
| --- | --- | --- |
| 料酒 | 1勺（15克） | 去腥 |
| 生抽 | 1勺（15克） | 调味，给底味 |
| 老抽 | 半勺（8克） | 上色 |
| 蚝油 | 1勺（15克） | 提鲜 |
| 白胡椒粉 | 半小勺（1克） | 去腥 |
| 玉米淀粉 | 1勺（15克） | 锁住肉汁，煎出来更嫩 |

**炒底料**（干锅的风味来源）

| 调料 | 用量 | 说明 |
| --- | --- | --- |
| 郫县豆瓣酱 | 1.5勺（22克） | 干锅的灵魂，炒出红油才有那个味 |
| 干辣椒 | 1小把 | 剪成段，辣度可以自己调 |
| 花椒粒 | 1小撮 | 增加麻香 |
| 大蒜 | 3-4瓣 | 切片或拍碎 |
| 姜 | 1小块 | 切片或切末 |
| 白糖 | 半小勺（2克） | 提鲜，平衡辣味 |
| 熟白芝麻 | 适量 | 出锅撒，增香好看 |

## 👨‍🍳 操作流程（香煎版）

1. **第一步：腌鸡翅、备配菜**

    + 鸡翅洗净，用厨房纸吸干水分。在两面各划两刀（或对半切开），方便入味。
    + 放入碗中，加入料酒、生抽、老抽、蚝油、白胡椒粉、淀粉，抓匀腌制20-30分钟。
    + 土豆去皮切粗条，莲藕切片，洋葱、青红椒切块。

2. **第二步：煎鸡翅、煎配菜（同时搞定）**

    + 煎制时用中小火，耐心是关键。

    1. 平底锅烧热，倒少许油（比平时炒菜略多），油热后转中小火（中小火）— 和煎椒盐鸡翅一样，火不能大
    2. 将鸡翅正面朝下平铺入锅（中小火，煎5-6分钟）— 不要急着翻动，等一面金黄定型再翻
    3. 翻面，煎另一面（中小火，煎4-5分钟）— 两面都煎到金黄焦脆，盛出备用
    4. 用锅中余油，放入土豆条和藕片（中小火，煎3-4分钟）— 煎到表面微焦、边缘金黄，盛出备用

3. **第三步：炒底料、合炒（和干锅花菜的炒法一样）**

    + 炒底料时用小火，把豆瓣酱的红油逼出来。合炒时转大火，快速翻炒。

    1. 锅中留底油，小火下姜、蒜、干辣椒、花椒，炒出香味（小火，约30秒）— 火别大，辣椒花椒糊了会发苦
    2. 下郫县豆瓣酱，小火慢炒（小火，约1分钟）— 炒出红油，这是颜色的来源，也是香气的关键
    3. 转大火，放入煎好的鸡翅、土豆、藕片，快速翻炒（大火，约1分钟）— 让所有食材都裹上红油
    4. 加白糖提鲜，放入洋葱和青红椒块（大火，约1分钟）— 炒到蔬菜断生即可，保持脆度
    5. 撒上白芝麻，出锅 — 注意：全程不加一滴水

## ✅ 成品标准

+ 颜色：红亮油润，鸡翅和配菜都均匀裹着酱色
+ 口感：鸡翅外皮焦香，内里嫩滑；土豆和藕片干香入味
+ 风味：麻辣咸香，复合味浓郁，是那种“一口接一口停不下来”的感觉
+ 盘底：干爽，没有汤汁，只有一层红油，这就是“干锅”的精髓

干锅鸡翅，煎鸡翅和干锅花菜两套技能合在一起了。鸡翅先煎香，配菜先煎熟，底料炒出红油，最后大火一锅端，全程不加一滴水，出来的就是干锅那种干香麻辣。
