# 椒盐鸡翅

干香、咸鲜、外焦里嫩，算是那种"不用配米饭都能空口吃好几个"的菜。

这道菜是干爽香酥、椒盐味直接裹在表面的，追求的是啃起来"咔嚓"一声的焦脆感。

## 💡 "椒盐"有两种玩法

椒盐鸡翅的成品状态，可以分成两种常见的做法，看你更喜欢哪种风格：

1. 先腌再煎，最后撒椒盐	

    + 操作方式: 鸡翅用生抽、蚝油、料酒、椒盐腌制，平底锅煎到两面金黄，出锅前再撒椒盐	
    + 成品特点: 外焦里嫩，椒盐香集中在表面，肉汁保留完整	
    + 最推荐，操作简单，少油烟，不用开油锅
2. 先腌再炸或裹粉炸	

    + 操作方式: 鸡翅腌制后裹面粉或淀粉，油炸或空气炸锅处理，出锅撒椒盐	
    + 成品特点: 外壳酥脆，香口更足，更适合下酒	
    + 想吃更酥脆的口感时用

推荐第一种：平底锅香煎版，用油少、不用裹粉、15-20分钟搞定。

## 🥩 食材清单（2-3人份，约8-10个鸡翅）

| 食材 | 用量 | 说明 |
| --- | --- | --- |
| 鸡翅中 | 8-10个（约500克） | 选个头均匀的，肉质饱满 |
| 姜 | 1小块 | 切片或切丝，腌制用 |
| 葱 | 1根 | 切段，腌制用 |
| 食用油 | 适量 | 煎鸡翅用 |

**腌料**

| 调料 | 用量 | 说明 |
| --- | --- | --- |
| 料酒 | 1.5勺（22克） | 去腥 |
| 生抽 | 1.5勺（22克） | 调味、上底味 |
| 蚝油 | 1勺（15克） | 提鲜 |
| 白胡椒粉 | 半小勺（1克） | 去腥增香 |
| 椒盐（腌鸡翅用） | 1小勺（3克） | 给鸡翅一个底味 |
| 玉米淀粉 | 1大勺（15克） | 锁住肉汁，煎出来更嫩 |

**出锅前撒的椒盐**：椒盐粉约1小勺（3克），出锅前撒，椒盐的麻香味才够冲。

| 洋葱 | 一小块 | 切小丁，出锅前炒香 |

## 👨‍🍳 操作流程（香煎版）

1. **第一步：处理鸡翅**

    + 鸡翅洗净，用厨房纸吸干表面水分。
    + 鸡翅一剪二：从鸡翅中间关节处剪开，一个鸡中翅变成两小块。这样更容易入味，煎的时候也更快熟透。
    + 如果不想剪开，也可以在鸡翅两面各划两刀，方便入味。

2. **第二步：腌制（关键）**

    1. 鸡翅放入大碗，加姜片、葱段、料酒、生抽、蚝油、白胡椒粉、1小勺椒盐
    2. 用手抓匀，让每块鸡翅都裹上腌料 — 抓匀后可以稍微按摩一下鸡翅
    3. 加1大勺玉米淀粉，再次抓匀 — 淀粉能锁住肉汁，煎出来外焦里嫩
    4. 盖上盖子，腌制至少30分钟 — 时间充裕的话腌1小时更入味

    + **关于椒盐的说明：** 腌的时候加一小勺椒盐是给鸡翅"打底味"，最后出锅前还要再撒一次椒盐，两次椒盐叠加，香味才够层次。

3. **第三步：煎鸡翅（中小火慢煎）**

    1. 平底锅烧热，倒油（比平时炒菜略多），油热后转中小火（中小火）— 火不能大，否则外焦里生
    2. 鸡翅正面朝下平铺入锅（中小火，煎5-6分钟）— 不要翻动，让表皮充分焦化
    3. 翻面，煎另一面（中小火，煎5分钟）— 煎到两面金黄、表面焦脆
    4. 如果鸡翅比较厚，可以盖盖焖2分钟（中小火，2分钟）— 帮助内部熟透，锁住肉汁

    + **判断标准：** 鸡翅两面金黄、用筷子能轻松戳透、没有血水渗出，就是熟了。
    + **不用放油的技巧：** 如果不粘锅性能好，鸡翅本身会出油，甚至可以不放油直接煎。

4. **第四步：撒椒盐出锅（如果加配菜）**

    1. 鸡翅煎好盛出，锅中留少许底油
    2. 下红绿辣椒丁和洋葱丁炒香（中小火，约30秒）
    3. 倒回鸡翅，撒1小勺椒盐（约20秒）— 快速翻炒，让鸡翅均匀裹上椒盐
    4. 出锅装盘，撒葱花点缀

    + **简化版：** 如果不想加配菜，鸡翅煎好后直接撒椒盐翻匀出锅就行。

## ✅ 成品标准

+ 外皮：金黄焦脆，表面微微起皱
+ 内里：嫩滑多汁，咬开有肉汁
+ 味道：椒盐咸香、麻味隐隐，蒜香和鸡油的香味融合
+ 盘底：干爽，无汤汁

## 💡 带饭专项

+ 适合带饭吗？	
    ⚠️ 一般。鸡翅的焦脆外皮回热后会变软，口感会打折扣
+ 微波回热	
    中高火1分30秒-2分钟，皮会变软但味道还在
+ 保鲜期	
    冷藏2天

椒盐鸡翅的**核心操作**就一句话：中小火慢煎，不要急着翻面，腌的时候放一次椒盐，出锅前再撒一次。这样做出来外皮焦脆、内部嫩滑多汁，椒盐的咸香和鸡肉的鲜香融合在一起
