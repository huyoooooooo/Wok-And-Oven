# 蒜香鸡翅

椒盐鸡翅靠的是椒盐的咸麻，而蒜香鸡翅是靠大量蒜蓉的浓郁香气渗进鸡肉里，吃到嘴里满口蒜香。

蒜香鸡翅有两种主流做法，正好对应两种完全不同的成品状态：

1. 香煎/焖煮版	
    + 操作方式：鸡翅腌制后平底锅煎到金黄，加蒜末和少许水焖煮收汁	
    + 成品特点：蒜香浓郁，肉质软嫩多汁，带少许浓稠汤汁	
    + 家常下饭，操作简单，不用开油锅

2. 油炸/裹粉版	
    + 操作方式：鸡翅腌制后裹粉，炸到外皮酥脆	
    + 成品特点：外壳酥脆，蒜香更集中在表面，咬下去“咔嚓”一声	
    + 当零食、下酒，口感更香脆

推荐第一种——平底锅香煎焖煮版。操作简单、蒜香浓郁、肉质软嫩，关键是不需要起油锅炸。

## 💡 为什么蒜香鸡翅这么香？—— **核心在"双蒜法"**

做蒜香鸡翅，蒜的用法是有讲究的。“双蒜法”，可以把蒜香发挥到极致：

+ 第一道蒜（腌制用）	
    蒜末和鸡翅一起腌制，让蒜味渗进肉里	提供“底味”，鸡翅内部也有蒜香
+ 第二道蒜（炒制用）	
    鸡翅煎好后再撒一把新鲜蒜末，稍微煎一下	提供“表香”，出锅前蒜香更冲、更直接

这样内外结合，蒜香才够浓郁。两种蒜不能省其中任何一种，不然味道差一截。

## 👨‍🍳 操作流程（香煎焖煮版）

1. **第一步：处理鸡翅**

    + 鸡翅洗净，用厨房纸吸干表面水分（这一步和椒盐鸡翅一样重要，有水的话煎不出焦香）。
    + 鸡翅两面各划两刀，方便入味和加快成熟。
    + 如果鸡翅比较大，可以像椒盐鸡翅一样从中间关节处一剪二，更易熟透。

2. **第二步：腌制（让蒜味渗进肉里）**

    1. 鸡翅放入大碗，加入姜片、料酒、生抽、老抽、蚝油、白糖、白胡椒粉、盐
    2. 加入半头蒜的蒜末（第一道蒜），用手抓匀 — 抓匀后可以稍微按摩一下，让调料渗进去
    3. 盖上盖子或保鲜膜，腌制至少1小时 — 时间充裕的话腌2-3小时或冷藏过夜，蒜味更浓

3. **第三步：煎鸡翅（中小火慢煎）**

    1. 平底锅烧热，倒油（比平时炒菜略多），油热后转中小火（中小火）— 火不能大，否则外焦里生
    2. 鸡翅正面朝下平铺入锅（中小火，煎5-6分钟）— 不要急着翻面，等底面金黄定型再翻
    3. 翻面，煎另一面（中小火，煎4-5分钟）— 两面都煎到金黄

4. **第四步：加第二道蒜，焖煮收汁**

    1. 鸡翅两面煎好后，撒入剩余半头蒜的蒜末（第二道蒜）（中小火，约30秒）— 把蒜末煎香，蒜香味飘出来
    2. 加入少量开水（约50ml，大概没过鸡翅的1/3高度）— 不需要加太多水，主要是帮助蒜香融合和鸡翅熟透
    3. 盖上锅盖，中小火焖煮（中小火，约3-5分钟）— 让鸡翅内部充分熟透，同时吸收蒜香
    4. 开盖，转大火收汁（大火，约1分钟）— 收到汤汁浓稠、裹在鸡翅表面即可出锅
    5. 出锅，撒葱花点缀

## 成品标准

+ 颜色：鸡翅表面金黄微焦，裹着浓稠酱汁
+ 口感：外皮焦香微韧，内部软嫩多汁
+ 味道：蒜香浓郁，咸鲜微甜，吃完满口蒜香
+ 盘底：少量浓稠汤汁（不是水汪汪的）

**汤汁状态：** 这道菜是带少许汤汁的，不像椒盐鸡翅那样完全干爽。汤汁收浓后裹在鸡翅表面，拌饭也很香。

## 💡 带饭专项

+ 适合带饭吗？	
    ✅ 可以。焖煮版带汤汁，回热后鸡肉反而更入味，但外皮的焦脆感会变软
+ 微波回热	
    中高火2分钟，汤汁会让鸡翅保持湿润，不会干柴
+ 保鲜期	
    冷藏2天

这道菜和椒盐鸡翅的区别是——椒盐鸡翅追求“干香酥脆”，蒜香鸡翅追求“蒜香浓郁、软嫩多汁”。一个干爽、一个带汁，正好互补。
