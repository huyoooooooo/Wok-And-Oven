# 红烧鸡翅

和可乐鸡翅从外表上看很像，都是红亮油润、酱汁浓稠，但前者是传统路子，后者是“偷懒版”。

北方的版本偏咸鲜口，到了南方则更多地带着咸甜回甘。核心和灵魂——“炒糖色”。

## 💡 可乐鸡翅 vs 红烧鸡翅

这俩菜最让人分不清，但实际操作下来，区别其实很清晰：

| 对比项 | 可乐鸡翅 | 红烧鸡翅 |
| --- | --- | --- |
| 上色和甜味来源 | 可乐本身（自带焦糖色和糖分） | 冰糖或白糖 + 老抽（炒糖色是关键） |
| 风味特点 | 甜味中带有可乐的果酸味和焦糖香，味道相对固定 | 咸甜平衡，酱香浓郁。甜味和咸味可以根据自己口味调整 |
| 操作难点 | 省了炒糖色的步骤，对新手友好 | 炒糖色是核心难点，火候不到颜色不亮，过了就容易发苦 |
| 口味灵活性 | 口味比较固定，就是可乐味 | 灵活度高，可以咸鲜、可以回甜，可以加香料，也可以加辣椒 |

## 🥩 食材清单（2-3人份）

| 食材 | 用量 | 说明 |
| --- | --- | --- |
| 鸡翅中 | 10-12个（约500克） | 翅中最佳，表面划两刀，方便入味 |
| 冰糖（或白糖） | 1-2勺（15-30克） | 炒糖色和提鲜用，冰糖成品颜色更红亮 |
| 料酒 | 1-2勺（15-30克） | 焯水和焖煮去腥 |
| 生抽 | 1.5勺（22克） | 提鲜 |
| 老抽 | 半勺-1勺（8-15克） | 加深颜色，补色用 |
| 姜 | 3-4片 | 去腥增香 |
| 葱 | 1根 | 切段 |
| 蒜 | 3-4瓣 | 切片 |
| 八角 | 1-2个 | 增香，和红烧很搭，也可不放 |
| 食用油 | 适量 | 煎鸡翅用 |

## 👨‍🍳 操作流程（重点：炒糖色）

这一步做好了，成品颜色就是红亮诱人的，味道也更有层次感。

+ **焯水去腥：** 
    - 鸡翅洗净后冷水下锅，加姜片和料酒，煮开后撇去浮沫，捞出用温水冲净沥干。
    - 焯水是为了成品汤汁干净不浑浊。
+ **煎香鸡翅：** 
    - 鸡翅两面各划2-3刀，划到骨头
    - 锅烧热倒少许油，中小火把鸡翅煎到两面金黄。
    - 煎过的鸡翅外皮更香，炖煮时也不容易散。
+ **炒糖色（关键）：** 
    - 锅留底油，放入冰糖，用中小火慢慢搅动。
    - 看到冰糖融化 → 冒小泡 → 变成枣红色/琥珀色时，立刻把鸡翅倒进去，快速翻炒，让每一块鸡翅都均匀地裹上糖色。
+ **焖煮入味：** 
    - 鸡翅裹好糖色后，加入葱姜蒜、八角，淋入生抽、料酒，翻炒出香味。
    - 倒入热水，水量要没过鸡翅（千万别加凉水，热肉遇冷会收缩，口感就柴了）。
    - 大火烧开后转中小火，盖上盖子焖煮约20分钟。
+ **大火收汁：** 
    - 开盖后尝一下咸淡（生抽有咸味，盐先不放），转大火开始收汁。
    - 汤汁从稀变浓，要不停地翻动鸡翅，防止糊锅。收到汤汁浓稠、能挂在鸡翅上时，就撒点葱花出锅。

---

炒糖色如果火候没掌握好，要么颜色不够亮，要么糖会发苦。如果你觉得把握不大，可以用老抽来调色。

在步骤3（炒香料后），直接加入1勺老抽，和生抽一起翻炒上色，让鸡翅均匀沾上酱色。后续的焖煮和收汁步骤完全不变。

这样做颜色比炒糖色稍弱一点，但味道同样正，操作难度大大降低。

做这道菜相当于多学一门“炒糖色”的功夫。这个技能学会了，以后做红烧肉、红烧排骨都能用上。
