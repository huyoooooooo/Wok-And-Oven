# 黄豆焖鸡脚

江西菜里非常经典的一道家常硬菜。

这道菜的核心是“慢”，要把鸡脚炖到软烂脱骨、一抿就化，黄豆焖到粉糯吸汁，汤汁浓稠到能挂在鸡脚上，用来拌饭是绝配。

这道菜的味道走向很明确：咸鲜为主，香辣突出，酱香浓郁。江西菜的辣是那种后劲很足的鲜辣，香料的味道会渗进鸡脚的骨头缝里。


## 这道菜的“慢”在哪里？

做这道菜，时间主要花在两个地方：

1. 黄豆预处理	
    + 干黄豆必须提前浸泡过夜：黄豆不泡透，焖多久都很难粉糯，口感会偏硬
2. 鸡脚炖煮	
    + 需要长时间炖到软烂：鸡脚里的胶质需要足够的时间和温度才能释放出来，变成软糯的口感

关于“怎么炖”有两种主流方式：

1. 高压锅版	
    + 鸡脚和黄豆一起用高压锅压煮 8-12分钟	
    + 速度最快，能快速把鸡脚压软，保留原汤	
    + ✅ 省时间，推荐
2. 普通锅慢炖版	
    + 用普通锅小火慢炖 50分钟-1小时以上	
    + 传统做法，鸡脚更入味，胶质释放更充分	
    + ⚠️ 适合周末有空时做

原汤是这道菜的灵魂，炖出来的汤汁要留着，后面炒制收汁时用。

## 👨‍🍳 操作流程（高压锅快手版）

1. **第一步：处理鸡脚和黄豆**

    + 鸡脚：剪去指甲，对半剖开（更容易入味），冷水下锅，加姜片和料酒焯水，煮开后撇去浮沫，捞出用温水冲净。
    + 黄豆：提前一晚浸泡，泡到涨发饱满，洗净沥干。

2. **第二步：高压锅炖煮**

    + 这一步把鸡脚压软，同时把黄豆焖到粉糯。

    1. 将焯好的鸡脚、泡好的黄豆放入高压锅
    2. 加入姜片、八角、桂皮、香叶、花椒、干辣椒 — 香料一次性放足
    3. 加入生抽2勺、老抽1勺、料酒1勺、啤酒（可选）和适量清水 — 水量没过食材即可
    4. 高压锅上汽后压煮（8-12分钟）— 时间到后自然泄压
    5. 开盖，将鸡脚和黄豆捞出备用，炖煮的汤汁保留 — 原汤是灵魂，千万别倒掉

3. **第三步：炒制收汁**

    + 这是把味道做"浓"的最后一步，也是出香味的关键。

    1. 炒锅烧热倒油，下姜蒜末和小米辣圈爆香（中火，约30秒）
    2. （可选）加1勺辣椒粉炒出红油（中火，约20秒）
    3. 倒入压好的鸡脚和黄豆，快速翻炒均匀（中火，约30秒）
    4. 倒入炖煮的原汤，水量大概到食材的1/2-2/3高度（中火）— 原汤是这道菜味道浓郁的关键
    5. 加生抽、老抽、蚝油、白糖、白胡椒粉调味（中火）— 尝一下咸淡，盐最后再决定放不放
    6. 大火收汁，期间不停翻动防止糊锅（大火，约3-5分钟）— 收到汤汁浓稠、能挂在鸡脚上即可，留一些汤汁拌饭
    7. 出锅，撒葱花点缀

## 👨‍🍳 操作流程（普通锅版）

**核心变化**就两处：黄豆的处理和焖煮时间。

+ 黄豆：提前泡足时间，4小时起步

    普通锅焖煮的火力比高压锅温和得多，黄豆必须提前泡透。

    - 冷水浸泡（推荐）	
        至少4小时，最好过夜	黄豆充分吸水涨发，焖煮时才容易粉糯
    - 热水/开水浸泡	
        1-2小时	比冷水快，但需要焖的时间相应延长
    - 替代方法：冷锅不放油，小火把干黄豆炒到表面微裂、闻到香味，再加水浸泡10分钟。经过炒制的黄豆更容易焖软。

+ 焖煮时间：40-60分钟，水量要加足
    - 高压锅版只需要8-12分钟，普通锅版需要延长到40分钟到1小时。

**操作注意**

+ 水量	
    一开始就一次性加足，水要没过所有食材。普通锅水分蒸发比高压锅多，中途加水会稀释味道
+ 火候	
    大火烧开后，立刻转小火，保持微沸状态慢慢炖
+ 中途检查	
    每隔20分钟左右开盖看一眼，翻动一下防止粘底，顺便看水量够不够
+ 判断熟度	
    鸡脚筷子轻轻一戳就能穿透、肉能脱骨；黄豆颜色变深、一抿就化，就可以了

**操作流程**

和高压锅版的区别只在炖煮这一步，其他步骤都一样：

1. 黄豆提前泡4小时以上（或过夜）（4小时+）— 泡到完全涨发
2. 鸡脚剪去指甲，冷水下锅加姜片料酒焯水，撇浮沫，温水洗净（5分钟）— 去腥
3. 炒锅热油，爆香姜蒜、八角、桂皮、香叶、干辣椒、花椒（约30秒）
4. 下鸡脚翻炒，加生抽、老抽、料酒、蚝油、白糖翻炒上色（约2分钟）
5. 倒入泡好的黄豆，加入足量热水（没过所有食材），大火烧开 — 水一次加足
6. 转小火，盖盖焖煮（40-60分钟）— 中途翻动1-2次
7. 开盖，尝咸淡，转大火收汁到浓稠（3-5分钟）— 留一些汤汁拌饭
8. 出锅，撒葱花

## ✅ 成品标准

+ 鸡脚：软烂脱骨，筷子轻轻一夹就能撕开，一抿就化
+ 黄豆：粉糯绵软，吸饱了汤汁的鲜辣味
+ 汤汁：浓稠红亮，能厚厚地裹在鸡脚和黄豆表面，盘底有余汁
+ 风味：咸鲜香辣，辣味有层次，酱香浓郁，非常下饭

菜品味道浓郁，黄豆吸饱了鸡脚的胶质和汤汁的香辣，吃起来粉粉糯糯的，和鸡脚完全是两种不同的口感。注意汤汁不要收太干，留一些用来拌饭。
