# 毛豆烧鸡

酱香浓郁，鸡肉吸饱了汤汁，咬一口有爆汁感，毛豆清甜粉糯，最后的汤汁拿来拌饭是精髓。

和鸡胸版相比，换成鸡腿后最大的变化是：焖煮时间从5分钟延长到20-30分钟，鸡腿肉久炖不柴，毛豆也能彻底焖到软烂入味。

## 🥩 食材清单（2-3人份）

| 食材 | 用量 | 说明 |
| --- | --- | --- |
| 鸡腿（琵琶腿） | 3-4个（约400-500克） | 剁成块，带骨更香 |
| 毛豆米 | 200-250克 | 新鲜或冷冻均可 |
| 姜 | 3-4片 | 切片 |
| 大蒜 | 3-4瓣 | 拍碎或切片 |
| 干辣椒/小米辣 | 2-3个（可选） | 增加微辣风味 |
| 八角 | 1-2个（可选） | 增香 |

**调味料**

| 调料 | 用量 | 说明 |
| --- | --- | --- |
| 生抽 | 2勺（30克） | 咸鲜底味 |
| 老抽 | 半勺-1勺（8-15克） | 上色用 |
| 蚝油 | 1勺（15克） | 提鲜增稠 |
| 白糖/冰糖 | 小半勺（2-3克）或几颗 | 提鲜回甜 |
| 料酒 | 1勺（15克） | 去腥 |
| 盐 | 最后尝味放 | 生抽蚝油已有咸度 |
| 热水 | 没过食材 | 焖煮用 |

## 👨‍🍳 操作流程（鸡腿版）

1. **第一步：处理食材**

    + 鸡腿剁块，洗净沥干。如果怕腥，可以冷水下锅焯水，加姜片和料酒，煮开后撇浮沫，捞出用温水冲净沥干。
    + 毛豆米洗净，如果用的是冷冻毛豆，直接解冻备用；新鲜毛豆可以先焯水2分钟去生味。

2. **第二步：炒鸡块**

    + 锅烧热倒油，下姜、蒜、干辣椒、八角爆香（中火，约30秒，八角增香，不习惯可以不放）
    + 下鸡块，煎到表面金黄、鸡皮微微卷起（中火，约3-4分钟，煎出鸡油，肉质更香）
    + 加料酒、生抽、老抽、蚝油、白糖翻炒上色（约1分钟，让每块鸡肉裹上酱色）

    **关于炒糖色：** 部分做法会在这一步用冰糖炒糖色再下鸡块，成品颜色更红亮。如果你嫌麻烦，直接用老抽上色效果也很好。

3. **第三步：焖煮**

    + 加入热水，水量没过鸡块（大火，一定要加热水，热肉遇冷水会收缩变柴）
    + 大火烧开，转中小火，盖上盖子焖煮（中小火，约15-20分钟，让鸡肉入味变软）
    + 放入毛豆米，翻炒均匀（中小火，继续焖煮8-10分钟，毛豆焖到软烂即可）

4. **第四步：收汁出锅**

    + 开盖，尝一下咸淡（生抽和蚝油已有咸度，盐最后再决定放不放）
    + 转大火收汁，不停翻动防止糊锅（大火，约2-3分钟，收到汤汁浓稠、能挂在鸡块上）
    + 撒葱花，出锅（保留一些汤汁拌饭）

## ✅ 成品标准

+ 鸡腿肉	嫩滑入味，咬开有汁水，轻松脱骨
+ 毛豆	粉糯软烂，吸饱了汤汁的酱香
+ 汤汁	浓稠红亮，裹在食材表面，盘底有余汁
+ 风味	咸鲜回甜，毛豆清甜和鸡肉酱香融合

---

和葱烧鸡腿操作逻辑类似——煎香→加液焖煮→收汁，区别是焖煮时间延长（鸡腿需要炖透，毛豆需要焖软）
