# 香煎巴沙鱼

核心：水分处理干净和煎出焦香不散

## 👨‍🍳 冷冻巴沙鱼柳的处理流程

1. **第一步：解冻（两种方式，看时间）**

    + 冷藏解冻（推荐）：提前一晚从冷冻室移到冷藏室（约8-12小时，第二天要做，提前准备）
    + 冷水浸泡解冻：连包装袋泡在冷水中（不要拆袋），不要用温水或热水（约1-2小时，临时决定做，时间紧张）

2. **第二步：吸干水分（重中之重）**

    + 解冻后拆袋，取出鱼柳
    + 用厨房纸反复吸干表面水分（吸到纸不湿为止，这道菜能不能煎出焦香，全看这一步）
    + 如果鱼柳很厚，可以从中间片开，变成两片薄一点的鱼排（更薄更容易煎透，也更容易入味）

3. **第三步：腌制（去腥+锁水）**

    | 调料 | 用量（1条鱼柳约200-250克） | 说明 |
    | --- | --- | --- |
    | 料酒 | 1勺（15克） | 去腥 |
    | 盐 | 小半勺（2克） | 底味 |
    | 白胡椒粉 | 小半勺（1克） | 去腥 |
    | 姜片 | 2-3片 | 腌制时铺在鱼柳上 |

    腌制时间：15分钟，不要太久，否则鱼肉会出水变软。

4. **第四步：煎前处理（再次吸干）**

    + 腌制完成后，鱼柳表面会析出一些水分，用厨房纸再吸干一次，然后两面拍上一层薄薄的淀粉（玉米淀粉或土豆淀粉都可以，薄薄一层就行，不用裹太厚）。

## 👨‍🍳 煎制流程

+ 平底锅烧热，倒油（比平时炒菜略多，约2勺/30克）（中火，油热到微微冒烟）
+ 鱼柳轻轻滑入锅中，不要翻动（中火，煎3-4分钟，等底面煎到金黄定型，再翻面）
+ 翻面，煎另一面（中火，煎2-3分钟，煎到两面金黄、外壳焦脆）

## 🎛️ 调味选择（三选一）

+ **选择A：椒盐版** 

    直接撒椒盐粉，和椒盐鸡翅一样简单直接。煎好后在锅里撒入椒盐，翻匀即可出锅，表面焦脆干香，是最省事的吃法。

+ **选择B：照烧汁版（需要调汁）**

    | 调料 | 用量 | 说明 |
    | --- | --- | --- |
    | 生抽 | 1.5勺（22克） | 咸味 |
    | 蚝油 | 1勺（15克） | 提鲜 |
    | 老抽 | 半勺（8克） | 上色 |
    | 白糖 | 1小勺（3克） | 提鲜回甜 |
    | 清水 | 2勺（30克） | 稀释 |

    鱼柳两面煎好后，倒入调好的照烧汁，转小火煮约1分钟，让汤汁收浓裹在鱼柳表面即可出锅。

+ **选择C：黑椒版（和菠萝黑椒牛肉粒一样调法）** 

    煎好后在锅里撒上现磨黑胡椒碎，翻匀出锅。也可以像照烧汁一样，用生抽+蚝油+黑胡椒+少许水调成酱汁，小火煮1分钟收浓。

## ✅ 成品标准

+ 外壳	金黄焦脆，微硬不软
+ 内部	洁白嫩滑，一筷子夹开有汁水但不散
+ 风味	椒盐咸香 / 照烧甜咸 / 黑胡椒辛香

## 💡 带饭提醒

+ 适合带饭吗？	
    ⚠️ 一般。香煎外壳回热后会变软，口感会打折扣。但味道还在
+ 微波回热	
    中高火1分30秒-2分钟
+ 保鲜期	
    冷藏1-2天
