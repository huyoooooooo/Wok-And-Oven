# 彩椒牛柳

和双椒鸡丁、滑炒肉片同一套节奏：腌→滑炒→炒配菜→回锅裹汁。区别在于牛肉比鸡胸肉更娇气，小苏打嫩化、滑炒变色即出是关键。

追求「牛肉嫩滑、彩椒脆甜、酱汁包裹」——全程大火、快节奏，总用时约 10-15 分钟（不含腌制）。

## 🥩 食材选择

**牛肉选哪个部位？**

| 部位 | 特点 | 适合 |
| --- | --- | --- |
| 牛里脊（首选） | 最嫩，无筋膜，贵但值得 | 追求嫩滑口感 |
| 牛腿肉（性价比） | 稍瘦，需要小苏打处理 | 预算有限时可用 |
| 牛柳（市场常见） | 介于两者之间，常用 | 市场方便购买 |

牛里脊肉价高但口感最佳；牛腿肉可以用，但腌制时要加 1/4 小勺小苏打帮助嫩化，腌制后冲洗干净再调味。

**彩椒怎么选？**

| 颜色 | 特点 |
| --- | --- |
| 红彩椒 | 甜度高、颜色亮 |
| 黄彩椒 | 清甜、带果香 |
| 绿彩椒 | 清爽、微脆 |

建议红 + 黄 + 半个绿，三种颜色搭配，成菜红黄绿相间，视觉上更有层次感。

**切法**：牛肉逆纹切条，约筷子粗细，长度 5-6cm。彩椒切条，和牛肉条大小接近。

## 🥩 食材清单（2-3 人份）

| 食材 | 用量 | 说明 |
| --- | --- | --- |
| 牛里脊/牛柳 | 250g | 逆纹切条，约筷子粗细 |
| 红彩椒 | 1 个 | 切条 |
| 黄彩椒 | 1 个 | 切条 |
| 绿彩椒 | 半个 | 切条 |
| 食用油 | 比平时炒菜略多 | 滑炒用 |

**腌牛肉料**

| 调料 | 用量 | 说明 |
| --- | --- | --- |
| 小苏打 | 1/4 小勺（约 1g） | 嫩化纤维；牛里脊可不加或少加 |
| 生抽 | 1 勺（15g） | 冲洗后加 |
| 蚝油 | 1 勺（15g） | 冲洗后加 |
| 料酒 | 1 勺（15g） | 冲洗后加 |
| 白胡椒粉 | 少许（1g） | 冲洗后加 |
| 玉米淀粉 | 1 勺（15g） | 上浆锁水 |
| 食用油 | 1 勺（15g） | 最后封油，防粘 |

用牛里脊的话小苏打可以不加或少加；牛腿肉必须加。

**调味汁（提前调好）**

| 调料 | 用量 |
| --- | --- |
| 生抽 | 1.5 勺（22g） |
| 蚝油 | 1 勺（15g） |
| 白糖 | 半小勺（2g） |
| 黑胡椒粉 | 小半勺（2g） |
| 淀粉 | 半勺（8g） |
| 清水 | 2 勺（30g） |

## 👨‍🍳 操作流程

1. **第一步：腌制牛肉（嫩滑的关键）**

    + 牛肉切条，加小苏打抓匀（牛腿肉必须加，牛里脊可省略）
    + 腌制 10 分钟后冲洗干净（洗掉小苏打，否则有涩味）
    + 加入生抽、蚝油、料酒、白胡椒粉、淀粉抓匀
    + 最后加食用油封住，防粘

2. **第二步：滑炒牛肉**

    + 锅烧热倒油，油量比平时略多（中大火，油温六成热）
    + 下牛肉条，快速划散（中大火，约 1 分钟）
    + 炒到牛肉变色即盛出，不要全熟

3. **第三步：炒彩椒**

    + 锅中留底油，下彩椒条（大火，约 1 分钟）
    + 炒到彩椒断生、颜色更亮，保持脆度

4. **第四步：回锅裹汁**

    + 倒回牛肉，快速翻炒均匀（大火）
    + 淋入调好的汁，快速翻炒（大火，约 20-30 秒）
    + 酱汁裹住牛肉和彩椒，汤汁变浓即可出锅

## ✅ 成品标准

+ 牛肉：嫩滑不塞牙，表面裹着薄薄酱汁
+ 彩椒：脆甜，颜色鲜艳，不过软
+ 酱汁：浓稠度刚好，挂在食材表面，盘底少量余汁
+ 风味：黑椒咸鲜，彩椒清甜，层次分明

## 💡 带饭专项

+ 适合带饭吗？
    ✅ 适合，回热后牛肉和彩椒口感保持得不错
+ 微波回热
    中高火 1 分 30 秒-2 分钟
+ 保鲜期
    冷藏 2 天
