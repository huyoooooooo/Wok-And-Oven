# 葱烧豆腐

操作上其实更省事，不用焯水、不用腌、不用等，十几分钟就能出锅，而且豆腐比肉更吸汁，葱香渗透进去之后，味道会非常浓郁。

## 🥬 豆腐选择

| 豆腐类型 | 适合度 | 说明 |
| --- | --- | --- |
| 老豆腐（北豆腐） | ✅ 首选 | 质地紧实，煎的时候不易碎，煎出来外焦里嫩 |
| 嫩豆腐（南豆腐） | ⚠️ 也可以 | 口感更嫩滑，但煎的时候容易碎，需要更小心操作 |

建议第一次用老豆腐，容错率高，煎出来的效果也更好。

关于切法，有搜索到的食谱提到：用手掰成小块，比刀切更容易入味，因为不规则边缘能更好地吸附酱汁。两种方式都可以，想入味更好就用手掰。

## 🥩 食材清单（2餐带饭量）
| 食材 | 用量 | 说明 |
| --- | --- | --- |
| 老豆腐 | 1块（约300-400克） | 切厚片或手掰成块 |
| 大葱 | 2根 | 葱白葱叶分开 |
| 大蒜 | 2-3瓣 | 切片（可选） |
| 食用油 | 2勺（30克） | 煎豆腐用 |

**料汁（提前调好）**

| 调料 | 用量 | 说明 |
| --- | --- | --- |
| 生抽 | 2勺（30克） | 咸味基底 |
| 老抽 | 半勺（8克） | 上色 |
| 蚝油 | 1勺（15克） | 提鲜 |
| 白糖 | 1小勺（3克） | 平衡咸味，提鲜 |
| 玉米淀粉 | 1勺（15克） | 勾芡，让汤汁裹住豆腐 |
| 清水 | 半碗（约100ml） | 调开酱汁 |
| 白胡椒粉 | 少许（1克） | 提味（可选） |

## 👨‍🍳 操作流程

1. 第一步：处理食材

    豆腐切厚片（约1-1.5cm厚）或手掰成块，用厨房纸吸干表面水分——这一步很重要，水多的话下锅会溅油，煎出来也不够焦香

    大葱葱白和葱叶分开——葱白切段（约4-5cm长），葱叶切葱花备用

料汁提前调好：生抽2勺 + 老抽半勺 + 蚝油1勺 + 白糖1小勺 + 淀粉1勺 + 清水半碗 + 白胡椒粉少许，搅拌均匀

2. 第二步：煎豆腐（中小火，别急）
    
    + 锅烧热倒油，油热后下豆腐块（中小火，豆腐下锅后不要立刻翻动）
    + 中小火约 2-3 分钟煎到一面金黄、表面有硬壳（轻轻晃动锅，豆腐能滑动就是定型了）
    + 翻面，中小火约 2-3 分钟煎另一面至金黄

3. 第三步：葱香入魂

    + 豆腐煎好后，把豆腐推到锅边，锅中间下葱白段，中小火约 30 秒-1 分钟，把葱白煎到边缘微焦、闻到浓烈葱香
    + 把豆腐和葱白混合，倒入调好的料汁（料汁下锅前再搅一下，淀粉容易沉底）
    + 盖上锅盖，中小火焖煮 2-3 分钟，让豆腐吸收葱香和酱汁
    + 开盖，转大火收汁约 30 秒-1 分钟，汤汁变浓稠、裹在豆腐上即可
    + 撒上葱花，出锅

## ✅ 成品标准
    
+ 豆腐	
    外皮金黄微焦，内部软嫩，咬开有汁水
+ 葱	
    葱白煎到焦香，吸饱了酱汁，本身也很好吃
+ 汤汁	
    浓稠挂勺，裹在豆腐表面，盘底有余汁
+ 风味	
    葱香浓郁，酱香回甜，咸鲜下饭

## 💡 关键细节（尤其注意这里）

| 细节 | 为什么重要 |
| --- | --- |
| 豆腐吸干水分再下锅 | 表面有水的话，煎不出金黄焦脆的外壳，还会溅油 |
| 下锅后不要立刻翻动 | 豆腐还没定型的时候翻动，很容易碎 |
| 葱白要先煎到焦香 | 葱的香气需要油脂和高温才能释放，不煎透就没有“葱烧”的灵魂 |
| 料汁提前调好 | 豆腐在锅里停留时间越短，越不容易碎 |
| 勾芡不能省 | 淀粉水让汤汁变浓稠，才能“抱”在豆腐上，而不是稀稀拉拉地流在盘底 |
