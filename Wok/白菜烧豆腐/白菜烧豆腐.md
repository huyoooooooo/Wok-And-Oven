## 可行性分析（饭店 vs 家常）

| 对比项 | 饭店做法 | 家常妥协方案（更优） |
| --- | --- | --- |
| 豆腐处理 | 大锅宽油炸至金黄 | 平底锅少油煎，效果接近且省油 |
| 白菜出水 | 先焯水再烧，保证卖相 | 不焯水，直接炒出水分，汤汁更鲜甜 |
| 关键火候 | 猛火快收汁 | 小火煨5分钟，让豆腐内部入味 |
| 增香秘诀 | 加猪油或高汤 | 煎豆腐的底油+半碗开水代替高汤 |

## 食材量化（2-3餐带饭量）

老豆腐（北豆腐）：1块（约400克）（嫩豆腐太软易碎且带饭回热会出水）。  

大白菜叶：5-6片（取中后部厚帮子，切斜刀片，更易入味）。  

配料：大蒜3瓣（拍扁）、干辣椒2个（可选）、小葱1根（切段）。  

调味汁（提前兑好）：生抽2勺 + 蚝油1勺 + 白糖小半勺（提鲜不甜） + 白胡椒粉少许（点睛之笔） + 清水半碗（约100ml）。  

## 制作流程（15分钟搞定）

1. 煎豆腐
    + 豆腐切1厘米厚片，厨房纸吸干表面水分（防溅油、易上色）。
    + 平底锅倒油，中火煎至两面金黄硬壳（约每面2分钟）。盛出备用。

    核心技巧：煎完豆腐的锅不要洗！这层焦香底油是整道菜的香味来源。

2. 煸白菜（出水）
    + 用煎豆腐的底油，中大火下蒜瓣和干辣椒爆香
    + 先下白菜帮子，中大火翻炒至边缘微微透明、变软（约1.5-2分钟）
    + 再下白菜叶子，翻炒至叶子塌秧变软（约1分钟）

    关键：此时白菜会出一些水，这是自然的，不要倒掉

3. 合体煨制（入味）
    + 放入煎好的豆腐，淋入提前兑好的调味汁，加入白菜叶。
    + 盖上锅盖，小火焖5分钟（豆腐内部会形成气孔吸汁）。
    + 开盖撒白胡椒粉和葱段，大火收汁10秒（别收太干，留一勺汤汁带饭）。

## 带饭与微波回热

| 问题 | 解决方案 |
| --- | --- |
| 豆腐回热后发干 | 装饭盒时保证每块豆腐都带一点汤汁，微波前无需额外加水。 |
| 白菜二次加热出水 | 装盒时豆腐铺下层，白菜盖上层，回热后白菜的水分会自然流到豆腐里，反哺入味。 |
| 最佳微波时间 | 中高火2分30秒，盖盖留缝。如果饭盒不带排气孔，中途打开搅拌一次。 |
| 绝配主食 | 配白米饭或杂粮饭，汤汁拌饭是一绝。 |

## 一些建议

白胡椒粉：超市必有，不可省略（去豆腥味的关键，比料酒管用）。

蚝油：如果没有，用半勺生抽+半勺糖代替，风味略降但依然好吃。

干辣椒：不放完全不影响，只是增加一丝风味层次。

因本人不吃肥肉，固未选择五花煸油，素菜荤油风味会更足，可临出锅淋几滴香油替换。
