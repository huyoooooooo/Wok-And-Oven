# 玉米排骨汤

核心只有一件事：把所有东西放进锅里，让它自己慢慢煮。

## 💡 核心逻辑

1. 排骨要焯水	
    冷水下锅煮开，撇掉浮沫，汤才清亮；焯水后用温水冲洗
2. 玉米要选对	
    用水果甜玉米，汤才会清甜
3. 水要一次性加足	
    中途加水会冲淡味道
4. 盐最后放	
    出锅前10-15分钟再加盐，放早了肉会柴
5. 时间要够	
    至少1小时，排骨才会软烂

## 🥩 食材清单（3-4人份，家庭汤量）
| 食材 | 用量 | 说明 |
| --- | --- | --- |
| 排骨（肋排） | 500克 | 斩成小段 |
| 甜玉米 | 1-2根 | 切段，选水果玉米 |
| 胡萝卜 | 1根 | 切滚刀块（可选） |
| 姜 | 4-5片 | 去腥 |
| 葱 | 1根 | 打结或切段 |
| 盐 | 最后尝味放 | — |
| 料酒 | 1-2勺（15-30克） | 焯水去腥用 |
| 白醋 | 几滴 | 炖汤时加，帮助钙析出 |

## 👨‍🍳 操作流程

1. 排骨斩成段，冷水下锅，加姜片和料酒（冷水下锅才能逼出血水）
2. 大火煮开，撇净浮沫（约3-5分钟，浮沫撇干净，汤才清亮）
3. 排骨捞出，用温水冲洗干净（不能用冷水，否则肉质收缩变柴）
4. 汤锅/砂锅中放入排骨、姜片、葱，加入最少没过食材表面3-5公分的水量（一次性加足，中途不加水）
5. 大火烧开，转小火慢炖（约40-50分钟，保持微沸状态）
6. 加入玉米段和胡萝卜块（玉米不怕煮，早放晚放都行）
7. 继续小火炖煮（约20-30分钟，炖到排骨筷子能轻松戳透）
8. 加盐调味（盐最后放，放早了肉柴）
9. 出锅，撒葱花

总用时：约1.5小时

## ✅ 成品标准

汤色：清亮微黄，不浑浊

排骨：筷子一戳就透，肉不塞牙

玉米：吸饱了肉汤，咬下去有汁水

味道：清甜、鲜香，不油腻

## 🍲 配料怎么加？

| 配料 | 作用 | 什么时候加 |
| --- | --- | --- |
| 胡萝卜 | 增加甜味和颜色 | 和玉米一起下锅 |
| 红枣、枸杞 | 增加清甜和营养 | 出锅前10分钟放枸杞即可 |
| 山药 | 增加浓稠度和营养 | 和玉米一起下锅 |
| 香菇 | 增加菌菇鲜味 | 和玉米一起下锅 |

第一次做，建议只加玉米和胡萝卜，先把原味的汤喝明白。
