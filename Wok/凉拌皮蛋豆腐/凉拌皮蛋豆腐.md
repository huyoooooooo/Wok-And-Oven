# 凉拌皮蛋豆腐

滑嫩冰凉、入口即化的豆腐搭配Q弹的皮蛋。夏天带饭或当开胃凉菜，都是绝配。

核心就两块：豆腐怎么完整脱模、皮蛋怎么切不粘刀

## 🥬 选材
| 食材 | 推荐选择 | 原因 |
| --- | --- | --- |
| 豆腐 | 内酯豆腐或嫩豆腐（南豆腐） | 口感细嫩滑爽，和皮蛋的Q弹形成对比 |
| 皮蛋 | 标注"无铅工艺"的松花蛋 | 更安全，风味也更正 |

豆腐选内酯豆腐最稳妥，嫩滑程度和口感都适合这道菜。

## 🥩 食材清单（2餐带饭量）

| 食材 | 用量 | 说明 |
| --- | --- | --- |
| 内酯豆腐 | 1盒（约300g） | 口感最嫩 |
| 皮蛋 | 2个 | 无铅工艺的松花蛋 |
| 大蒜 | 3-4瓣 | 切末，料汁基底 |
| 小米辣 | 1-2个 | 不吃辣可以换成菜椒或不放 |
| 香菜/小葱 | 适量 | 点缀增香，二选一即可 |

料汁配方（基础版）

| 调料 | 用量 | 说明 |
| --- | --- | --- |
| 生抽 | 2勺(30g) | 咸味基底 |
| 香醋 | 1-1.5勺(22g) | 中和皮蛋碱性，提鲜 |
| 蚝油 | 1勺(15g) | 提鲜（可选） |
| 白糖 | 半小勺(2-3g) | 平衡酸咸，提鲜 |
| 香油 | 1勺(15g) | 增香，不可缺少 |
| 盐 | 少许(1g) | 生抽和蚝油有咸味，最后尝味再放 |

料汁中用量具体根据个人口味酌情调整

## 👨‍🍳 完整操作流程

1. 处理豆腐

    实用小技巧：用剪刀在豆腐盒底部的四角各剪开一个小口，然后撕掉包装膜，倒扣在盘子上，就能完整脱模。

    用刀轻轻划成厚片或小块（不要切太碎）/（可选）焯水后放凉再摆盘

2. 处理皮蛋

    | 方法 | 操作 | 效果 |
    | --- | --- | --- |
    | 刀沾水法 | 准备一碗凉白开，每切一刀前刀沾一下水 | 最常用，效果好 |
    | 棉线切割法 | 用一根干净棉线代替刀，把皮蛋"勒"开 | 切面最完整，不粘任何东西 |

    摆放在豆腐旁边或上面

3. 调料汁
    大蒜切末，小米辣切圈

    小碗中放入蒜末、小米辣、生抽、香醋、蚝油、白糖、香油，搅拌均匀

4. 合体
    将料汁均匀淋在豆腐和皮蛋上、撒上香菜或葱花点缀

提升风味的一步（可选）：热油激香  
如果想要更浓郁的香气，可以在淋料汁前，把蒜末和小米辣堆在豆腐上，烧一勺热油（约20ml）淋上去，激发出蒜香和辣椒香，再加生抽、醋等调匀。

## 💡 带饭专项

+ 适合带饭吗？

  ✅ 非常适合。凉拌菜冷藏后风味更融合，夏天吃尤其舒服

+ 装盒建议

  料汁单独装小盒，吃之前再淋上去。提前拌好豆腐会出水，影响口感

+ 微波回热

  不建议加热，这道菜适合冷吃，冰凉滑嫩是它的特点

+ 保鲜期

  冷藏1天内吃完，时间长了豆腐会出水变软

+ 关键提醒

  凉拌菜不宜久放，建议现做现吃

## 🎛️ 偏好调整

| 变量 | 调整方向 |
| --- | --- |
| 豆腐品种 | 内酯豆腐最嫩，嫩豆腐（南豆腐）也可以，老豆腐不推荐 |
| 皮蛋是否煮过 | 煮过好切、不粘刀，不煮风味更直接 |
| 是否用热油泼 | 泼油更香，不泼更清爽 |
| 辣度 | 小米辣决定辣度，不吃辣可以换成菜椒或不放 |
| 醋量 | 醋可以中和皮蛋的碱性，建议至少放1勺，可根据口味增减 |
