# 凉拌巴沙鱼

不用看火候、不用煎到焦香，水煮熟、浇上酱汁、拌匀就行。它肉质嫩滑、没有刺，做成凉拌菜特别适合夏天。

做法高度统一，基本是“水煮+过凉+浇汁”这套流程

## 👨‍🍳 操作流程（凉拌版）

1. 第一步：**处理鱼柳**

    巴沙鱼柳解冻，用厨房纸吸干表面水分。

    切成大块或厚片，不要切太小，否则煮熟后一拌就碎了。

2. 第二步：**煮熟**

    + 烧一锅水，加姜片和1勺料酒（去腥）
    + 水开后下鱼块，用漏勺轻轻划散
    + 煮到鱼块变白、浮起（约1-2分钟，不要煮太久，否则鱼肉会散）
    + 捞出，立刻放入凉水/冰水中降温（和口水鸡泡冰水一样，让肉质紧实）
    + 捞出控干水分（水分不控干，酱汁会被稀释）

3. 第三步：**调酱汁**

    | 调料 | 用量 | 说明 |
    | --- | --- | --- |
    | 蒜末 | 3-4瓣（约10克） | — |
    | 小米辣 | 2-3个 | 切圈，辣度可调 |
    | 辣椒面/粗辣椒粉 | 1勺（5克） | 增加红油色和香辣味 |
    | 白芝麻 | 1勺（10克） | 增香 |
    | 生抽 | 2勺（30克） | 咸味基底 |
    | 陈醋/香醋 | 1-1.5勺（15-22克） | 酸味 |
    | 蚝油 | 1勺（15克） | 提鲜 |
    | 白糖 | 半小勺（2克） | 提鲜回甜 |
    | 凉白开 | 2勺（30克） | 稀释酱汁，不会太咸 |

    凉拌酱汁：蒜末+小米辣+辣椒面+白芝麻泼热油，加生抽、蚝油、陈醋、白糖、香菜（可选）

4. 第四步：组装
    
    将控干水分的鱼块放入碗中，撒上香菜段（喜欢就放，不喜欢可以不放）。

    淋入调好的酱汁。

    轻轻拌匀即可。

## ✅ 成品标准

+ 鱼肉	嫩滑、不散、没有腥味
+ 酱汁	酸辣开胃，裹在鱼块表面
+ 整体	冰凉清爽，适合夏天
