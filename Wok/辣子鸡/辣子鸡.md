# 辣子鸡

目标明确：鸡肉外酥里嫩、麻辣味足、干香不油。在干辣椒和花椒里找鸡肉吃

所以辣子鸡的成品，干辣椒应该能把鸡肉盖住，而不是零星几个辣椒点缀。

## 🍖 鸡肉选什么部位？

| 部位 | 适合度 | 说明 |
| --- | --- | --- |
| 鸡腿（琵琶腿） | ✅ 首选 | 肉质嫩、有弹性，炸完外酥里嫩，不柴 |
| 整鸡剁块 | ⚠️ 也可以 | 部位杂，口感不均匀，操作难度大 |
| 鸡胸肉 | ❌ 不推荐 | 瘦、干、柴，炸完容易硬邦邦 |

家常做辣子鸡，推荐用鸡腿（去骨），操作简单，口感也最好。

## 🥩 食材清单（2-3人份）

| 食材 | 用量 | 说明 |
| --- | --- | --- |
| 鸡腿（琵琶腿） | 4个（约500克） | 去骨，切成2-3cm丁 |
| 干辣椒 | 80-100克 | 剪成段，去籽 |
| 花椒 | 1把（约10克） | 四川汉源花椒为佳 |
| 大蒜 | 4-5瓣 | 切片 |
| 姜 | 1小块 | 切末 |
| 大葱 | 1根 | 切段 |
| 熟白芝麻 | 适量 | 出锅撒 |

**腌肉料**

| 调料 | 用量 | 说明 |
| --- | --- | --- |
| 料酒 | 1勺（15克） | 去腥 |
| 生抽 | 1勺（15克） | 调味 |
| 盐 | 半小勺（2克） | 腌制时放足，炸后盐味进不去 |
| 白胡椒粉 | 半小勺（1克） | 去腥 |
| 玉米淀粉 | 2勺（30克） | 裹粉，让炸出来更酥脆 |
| 食用油 | 1勺（15克） | 封油防粘 |

**炒制调料**

| 调料 | 用量 | 说明 |
| --- | --- | --- |
| 食用油 | 足量（炸鸡用） | 能没过鸡肉的量 |
| 白糖 | 半小勺（2克） | 提鲜平衡 |
| 生抽 | 半勺（8克） | 出锅前调味 |

## 👨‍🍳 操作流程

1. 第一步：处理鸡肉

    鸡腿去骨，切成2-3cm的小丁（大小要均匀，太小容易炸干，太大不易入味）

    加入料酒、生抽、盐、白胡椒粉、淀粉、食用油，抓匀腌制20分钟

2. 第二步：处理干辣椒

    干辣椒剪成段，去掉大部分籽（籽容易炸糊发苦）

    可用开水泡10分钟后沥干，再炸时不容易糊

3. 第三步：炸鸡肉

    1. 锅烧热倒足量油，油温六成热（筷子插入冒泡）（中大火，油温要高，否则鸡肉炸干不酥）
    2. 下鸡丁，定型后划散（中大火，约3-5分钟，炸到表面金黄酥脆）
    3. 捞出，油温升到七成热（大火）
    4. 复炸1分钟（大火，1分钟，捞出控油）

鸡丁下锅后，油面应该完全没过鸡肉;复炸是外酥里嫩的关键，不能省。

4. 第四步：炒制（全程注意火候）

    1. 锅中留底油，中小火下姜蒜末炒香（中小火，20秒）
    2. 下花椒，炒出麻香（中小火，20秒，花椒先炒，出香慢）
    3. 下干辣椒段，炒到颜色变深、香气呛鼻（中小火，约30秒，火不要大，否则辣椒糊了发苦）
    4. 下炸好的鸡丁，转大火快速翻炒（大火，30秒）
    5. 加白糖和少许生抽，快速翻匀（大火，10秒，尝一下咸淡再决定是否加盐）
    6. 下大葱段，撒白芝麻，翻匀出锅（大火，20秒）

炒辣椒和花椒时务必小火，否则糊了会发苦，整道菜就毁了。

---

第一次炸（六成热）：约170-180℃，把鸡丁炸熟炸酥

复炸（七成热）：约190-200℃，让外壳更酥脆、逼出多余油分

如果对油温把握不准的，可以使用探针式温度计

## ✅ 成品标准

鸡肉	外皮酥脆，内部嫩滑，干香但不柴

辣味	麻辣味足，辣而不燥

干辣椒	颜色棕红，没有焦黑，能闻到呛辣香

整体	干爽无汤汁，辣椒把鸡肉盖住

---

炸鸡丁时，不要一次性把所有鸡丁都倒进去。分批下锅，好处：

    油温不会骤降

    每一批都能炸得酥脆

    用油量也可以少一些（因为不用一次性塞满锅）

炸完所有鸡丁之后，剩下的油如果量还不少

    让油自然冷却到室温（不要用水冲、不要放冰箱急冷）

    用细漏勺或厨房纸巾+滤网，把油里的残渣过滤掉（辣椒碎、花椒碎、面糊渣）

    装入干净的玻璃瓶或密封罐中，标注“炸过辣子鸡”，放冰箱冷藏保存

下次炸东西时优先用这瓶油，或者用来炒菜（尤其是炒有辣味的菜，比如 蒜蓉炒青菜、干锅花菜 等风味更足）会非常香，因为里面有鸡油的香味和花椒辣椒的底味
