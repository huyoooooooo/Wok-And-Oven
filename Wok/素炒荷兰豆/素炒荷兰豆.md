# 素炒荷兰豆（清甜脆嫩版）

一道简答的快手菜

## 食材清单（2餐带饭量）
| 食材 | 用量 | 说明 |
| --- | --- | --- |
| 荷兰豆 | 250克 | 一盒的量，撕去两侧老筋，洗净沥干 |
| 大蒜 | 3-4瓣 | 切片 |
| 干辣椒 | 2-3个 | 不吃辣可省略 |
| 盐 | 半小勺（2克） | 出锅前放 |
| 白糖 | 1小勺（3克） | 提鲜，吃不出甜味 |
| 食用油 | 2勺（30克） | 炒菜用 |

## 操作流程

1. 荷兰豆撕去两侧老筋，洗净沥干；大蒜切片，干辣椒剪段
2. 锅烧热倒油，锅要烧到冒青烟再倒油
3. 转中小火，下蒜片和干辣椒	中小火(15-20秒)炒到蒜片边缘微黄、闻到香味，不要炒糊
4. 转大火，下荷兰豆快速翻炒	大火(1-1.5分钟)不停翻动，受热均匀
5. 炒到荷兰豆颜色变深绿、表面微微起皱(断生)
6. 加盐和白糖(出锅前放，避免蔬菜提前出水)，快速翻炒均匀，大火(约20秒)炒匀出锅

## 成品标准

| 判断维度 | 理想状态 |
| --- | --- |
| 颜色 | 翠绿中带亮泽，不是暗绿 |
| 口感 | 脆嫩爽口，咬下去有"咔"的一声 |
| 味道 | 清甜，蒜香打底，能吃出荷兰豆本身的鲜甜 |
| 盘底 | 有少量余油，但荷兰豆表面不油汪汪 |

## 带饭专项

+ 适合带饭吗？	
    ✅ 可以，但要接受回热后脆度会略微下降（从“咔”变成“韧”）
+ 油多怎么装盒？	
    装盒前把菜夹起来装盒，锅底的余油不要倒进去。这样菜本身带一点油光，但饭盒底部不会积一层油
+ 微波回热
    中高火1分30秒-2分钟，盖盖留缝
+ 保鲜期
    冷藏1-2天，时间长了荷兰豆会变黄变软
