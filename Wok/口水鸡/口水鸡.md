# 口水鸡

和凉拌鸡丝、椒麻鸡丝凉面一样，都是冷吃的鸡肉凉菜，灵魂在于红油和复合酱汁，麻辣鲜香、回甜微酸，鸡肉嫩滑带脆。

但比凉拌鸡丝多两个新动作：冰水浸泡让鸡皮变脆，以及做一份香辣红油

## 👨‍🍳 核心步骤

1. **煮鸡腿**：时间要准，不能煮过头

    带骨鸡腿中火煮15分钟后再焖10分钟，肉刚熟、骨头还带一点粉红色时口感最好。

    + 鸡腿冷水下锅，加姜片、葱段、料酒、花椒（去腥增香）
    + 大火煮开，转中火煮（15-20分钟，视鸡腿大小调整）
    + 关火，盖上盖子焖（10-15分钟，让余温把鸡肉焖透）
    + 捞出，立刻放入冰水中浸泡（10-15分钟，让鸡肉紧实，鸡皮变脆）

    冰水浸鸡的步骤不能省，这是鸡肉嫩滑、鸡皮脆爽的关键。

2. **调酱汁**：红油是灵魂

    配方（2个鸡腿的量）：

    | 调料 | 用量 | 说明 |
    | --- | --- | --- |
    | 生抽 | 4勺(60g) | 咸味基底 |
    | 香醋 | 1-2勺(22g) | 提酸 |
    | 白糖 | 1小勺(3g) | 提鲜回甜 |
    | 盐 | 1小勺(3g) | — |
    | 红油/辣椒油 | 4大勺(60g) | 风味来源，可以买现成的 |
    | 花椒油/藤椒油 | 2大勺(30g) | 增加麻香 |
    | 蒜末、葱花、花生碎 | 适量 | 撒面用 |

    关于红油：可用市售的辣椒油或红油替代，味道会稍弱一些。家常做法和椒麻鸡丝凉面的泼油类似——油热后关火稍降温，倒入辣椒面+芝麻+花椒粉中搅匀。

3. 码盘浇汁
    
    鸡肉冰镇后捞出，斩成块，码在盘中（可以垫一层黄瓜丝打底）。酱汁充分搅匀后淋在鸡肉上，撒花生碎、葱花、白芝麻。

## 📋 操作节奏（用鸡腿做）

1. 鸡腿冷水下锅，加姜葱料酒花椒煮（15-20分钟）
2. 关火焖（10-15分钟）
3. 捞出泡冰水（10-15分钟）
4. 调酱汁（生抽+醋+糖+红油+花椒油+蒜末）（约5分钟）
5. 鸡腿切块摆盘，浇酱汁，撒花生碎葱花（约5分钟）

总用时：约40-50分钟（含浸泡）
